# 初期シータデータ [IUT I, Definition 3.1]

---

## 定義

初期シータデータとは以下の組:

```
(F̄/F, E_F, l, M̄, V, V^bad_mod, ε)
```

### 各成分の厳密な定義

| 記号 | 定義 | 制約条件 |
|------|------|---------|
| F | 数体 | — |
| F̄ | F の代数閉包 | — |
| E_F | F 上の楕円曲線 | F のすべての非アルキメデス付値で安定還元 |
| l | 素数 | l ≥ 5 |
| F_mod | E_F の定義体 (field of moduli) | F_mod ⊆ F |
| K | K := F(E_F[l])（l-等分点の定義体）| Gal(K/F) ⊇ SL₂(F_l) |
| V | K の付値の部分集合 | 自然な包含 F_mod ⊆ K が V ≅ V_mod を誘導 |
| V_mod | F_mod のすべての付値の集合 | — |
| V^bad_mod | V_mod の部分集合 | E_F が乗法的悪い還元を持つ奇数剰余標数の付値、非空 |
| C_K | C_K := C_F ×_F K | K-core であること |

### 導出される記号

```
l* := (l-1)/2
V^bad := V^bad_mod ×_{V_mod} V
V^good_mod := V_mod \ V^bad_mod
V^good := V \ V^bad
V^non := V の非アルキメデス付値
V^arc := V のアルキメデス付値
```

### 初期シータデータから決まるもの

1. **被覆 C̄_K → C_K**: 次数 l の有限エタール被覆。E_K[l] → Q (≅ Z/lZ) という階数1商から決まる
2. **X̄_K := C̄_K ×_{C_F} X_F**: 穴あき楕円曲線の被覆
3. **カスプ ε**: C̄_K のカスプ。V^bad における Q の標準生成元（双対グラフの唯一のループから決まる）に対応

---

## ガロア群の条件の意味

**Gal(K/F) ⊇ SL₂(F_l)** の条件は:

- E_F[l] の F 上のガロア表現が「十分に大きい」像を持つことを保証
- Serre の開像定理により、ほとんどの楕円曲線で十分大きな l に対して自動的に成立
- この条件は理論の随所で F_l 上の対称性（加法的・乗法的）を利用するために必要

---

## 初期シータデータの「存在」

ABC予想を証明するためには、任意の楕円曲線 E/F と十分大きな l に対して
初期シータデータの条件を満たすことが必要。

これは [GenEll] の理論により保証される:
- 任意の E/F に対し、適切な有限拡大と l の選択で条件を満たすことができる
- 最終的な不等式は、拡大の次数に依存しない形で正規化される
