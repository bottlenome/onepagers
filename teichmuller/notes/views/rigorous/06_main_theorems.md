## IUT III 定理A: マルチラジアル分裂モノイドの存在

**参照**: [IUT III, Theorem A]

### 主張

対数テータ格子上で、Θ^{×μ}_{LGP}-リンクに関する
LGP-モノイドの**分裂モノイド**を構成するマルチラジアルアルゴリズムが存在する。

### 技術的内容

対数テータ格子の格子点 (n,m) において:
1. 垂直列の log-リンク反復から対数殻を構成
2. 水平列の Θ-リンクから LGP-モノイドを構成
3. テータ関数の剛性定理と共役同期を用いて、
   LGP-モノイドの「分裂」（テータ値と q-パラメータの分離）を実現
4. この分裂がマルチラジアル（異なる算術的正則構造からも well-defined）

## IUT IV 定理1.10: 明示的ディオファンタス不等式

**参照**: [IUT IV, Theorem 1.10]

### 主張

「十分に一般的な位置」にある楕円曲線 E_F（初期シータデータの条件を満たすもの）に対し:

```
(1/6) · log(q_E) ≤ (1 + ε_0) · (log(d_{F_tpd}) + log(f_{F_tpd})) + C(ε_0, l)
```

ここで:
- q_E: 楕円曲線の q-不変量の対数（高さに相当）
- d_{F_tpd}: 数体 F_tpd の絶対判別式
- f_{F_tpd}: 楕円曲線の導手
- ε_0: l のサイズに依存する小さな量
- C(ε_0, l): 明示的な定数

### IUT III からの導出

IUT III 定理B / 系3.12 の抽象的な対数体積の見積もりを:
1. 各非アルキメデス付値でのテンソル積の対数体積に翻訳（命題1.1–1.4）
2. アルキメデス付値での見積もり（命題1.5）と結合
3. 素数分布の古典的定理（命題1.6）を適用
4. 明示的な不等式を得る

## IUT IV 定理A (= Corollary 2.3): 一般的ディオファンタス不等式

**参照**: [IUT IV, Corollary 2.3]

### 主張

X: 数体上の滑らかな射影曲線
D ⊆ X: 被約因子
U_X := X \ D: 双曲的曲線
d: 正整数
ε ∈ R_{>0}: 正の実数
ω_X: X の標準層

このとき:

```
ht_{ω_X(D)} ≤ (1 + ε)(log-diff_X + log-cond_D)
```

が U_X(Q̄)^{≤d} 上の関数の bounded discrepancy class として成立。

### "bounded discrepancy class" の意味

関数 f, g に対し f ≤ g (bd. disc.) とは:

```
∃ C > 0  s.t.  f(P) ≤ g(P) + C  ∀ P ∈ U_X(Q̄)^{≤d}
```

定数 C は ε, d, X, D に依存するが P には依存しない。

### 各項の定義

| 記号 | 定義 | 意味 |
|------|------|------|
| ht_{ω_X(D)} | ω_X(D) に関する標準高さ | 曲線上の点の「大きさ」|
| log-diff_X | 定義体の判別式の対数 | 数体の分岐の度合い |
| log-cond_D | 因子 D の導手の対数 | 悪い還元の度合い |
| U_X(Q̄)^{≤d} | 次数 ≤ d の代数的点全体 | 点の定義体の次数制限 |

### 定理1.10からの帰着

[GenEll] の理論を用いて:
- 任意の楕円曲線 → 「十分に一般的な位置」の楕円曲線に帰着
- 楕円曲線に対する定理1.10 → Vojta型不等式（定理A）

## ABC予想への特殊化

定理Aで X = P¹, D = {0, 1, ∞} とすると:

- U_X = P¹ \ {0, 1, ∞}
- ω_X(D) の次数 = 1（双曲的）

a + b = c (gcd(a,b) = 1) に対応する点 P = [a : c] ∈ P¹ \ {0, 1, ∞} を考えると:

```
ht_{ω_X(D)}(P) ∼ log max(|a|, |b|, |c|) = log c
log-cond_D(P) ∼ log rad(abc)
log-diff_X(P) ∼ 定数（有限個の例外を除く）
```

したがって:

```
log c ≤ (1 + ε) log rad(abc) + C
```

⟺  c ≤ K(ε) · rad(abc)^{1+ε}

これが **ABC予想**。
