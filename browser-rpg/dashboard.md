# Loveless Chronicle

転職・スキル・自動戦闘が楽しめるRPGブラウザゲーム。

## 概要

- 種別: game
- 状態: wip
- ファイル: `index.html` (ビルド生成物)
- ビルド: `bash build.sh`

## ソース構成

```
src/
├── css/style.css          # スタイル
├── data/                  # ゲームデータ
│   ├── constants.js       # 定数
│   ├── jobs.js            # 職業
│   ├── skills.js          # スキル
│   ├── equips.js          # 装備
│   ├── items.js           # アイテム
│   ├── monsters.js        # モンスター
│   ├── areas.js           # エリア・マップ
│   ├── shops.js           # ショップ
│   ├── recipes.js         # 合成・鍛冶レシピ
│   └── arena.js           # 闘技場
├── engine/                # ゲームロジック
│   ├── state.js           # 状態管理
│   ├── stats.js           # ステータス計算・成長
│   ├── combat.js          # 戦闘システム
│   ├── economy.js         # 経済(ショップ・銀行・倉庫)
│   ├── crafting.js        # クラフト(鍛冶・合成・強化)
│   ├── navigation.js      # ナビゲーション
│   └── save.js            # セーブ・ロード
├── ui/                    # UI
│   ├── components.js      # 共通コンポーネント
│   ├── screens.js         # 画面レンダリング
│   ├── events.js          # イベントハンドリング
│   └── render.js          # レンダラー
├── plugins/social.js      # ソーシャルプラグイン(スタブ)
└── main.js                # エントリポイント
```

## 機能

- [x] タイトル画面・名前入力・職業選択
- [x] 10種の職業 (下級4 + 上級6)
- [x] 転職システム (ステータス50%引き継ぎ, 上級職転職時ステータスボーナス, 上級職→上級職は不可)
- [x] 隠し成長確率システム
- [x] 装備の成長率ボーナス
- [x] 自動戦闘 (3つの作戦: 力技/呪文/万能 + 戦闘中切替, 敵DEF/MDEF耐性による三すくみ)
- [x] スキルシステム (職業+装備+鍛冶付与)
- [x] 9エリア・6町のワールドマップ (町の外は必ずモンスター出現階層に接続, 荒野の道20→古城21接続)
- [x] 5体のボス (転職ループ基準で難易度設計: ドラゴン2ループ, エルフ王4ループ(魔法特化ボス/戦士↔僧侶交互→ルーンナイト), 古城の主4ループ, 魔神10ループ, 伝説のプレイヤーたち20ループ+装備+5)
- [x] 武器耐久値・修理
- [x] 強化システム (+1~+10, 失敗で破壊)
- [x] 合成システム (レシピ制, 魔の森素材→魔王装備)
- [x] 鍛冶システム (鉱石→装備, 稀に貴重スキル付与, オリハルコン・古城・深層装備対応)
- [x] 闘技場 (NPC戦)
- [x] 銀行・倉庫
- [x] 宿屋 (全6町に設置)
- [x] 経験値システム (高階層ほどボーナス, 低階層ほどペナルティ)
- [x] セーブ/ロード (localStorage)
- [x] プラグインAPI (ソーシャル拡張用)
- [x] デバッグキャラクター (特定名入力でLv30テストキャラ: DRAGON, ELFKING, CASTLE, DEMON, LEGEND)
- [x] ダンジョン限定ドロップ (ラストダンジョン→オリハルコン, 魔の森→闇の結晶+精霊石, 古城→呪われた骨, 古城深層→幻影の布)
- [x] オープニングデモバトル (Lv30騎士vsドラゴン戦を体験後に冒険開始)
- [x] 転職ループの可視化 (転職回数表示, NPC台詞ヒント, 転職後初回レベルアップメッセージ)
- [x] 成長率演出 (ボーナス成長を金色★表示, 職業説明文に成長傾向ヒント)
- [x] 下級職パッシブ (戦士:被ダメ10%軽減, 魔法使い:毎ターンMP1回復, 僧侶:勝利後HP10%回復, 盗賊:ドロップ率+5%)
- [x] 死亡ペナルティ (所持金15%喪失, 銀行預金は安全)
- [x] 冒険者ギルド (始まりの町, ボス討伐リスト表示)
- [x] 装備比較UI (ショップ購入時に現装備との差分表示)
- [x] 転移の巻物 (消耗品アイテム, 探索済み最深階層へワープ)
- [x] 初回格上モンスター遭遇 (ゲーム開始直後にLv8強敵と強制遭遇)
- [x] 必殺技システム (1戦闘1回, 最強スキルMP不問発動, 発動後被ダメ2倍のリスク)
- [x] 共鳴システム (装備×作戦で発動する特殊効果。猛る刃/魔力の奔流/反撃の構え/一閃の連鎖の4種。共鳴限定ドロップ素材4種)
- [ ] 酒場チャット (プラグイン)
- [ ] 対人闘技場 (プラグイン)
