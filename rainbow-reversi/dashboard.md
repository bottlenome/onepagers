# 虹色リバーシ

虹色グラデーションの駒で最大4人対戦できるリバーシ。

## 概要

- 種別: game
- 状態: done
- ファイル: `index.html`

## ルール

- 6×6 / 8×8 / 10×10 ボード（デフォルト 8×8）
- 2〜4人対戦
- 全員挟み（自分の駒で挟んだ間の駒を色問わず自分の色に変える）
- 置けないプレイヤーはパス（全員置けなくなったら終了）
- 初期配置: 中央に各プレイヤー2個ずつ、円形に交互配置（ランダム回転）

## 機能

- [x] 可変サイズボード描画（6×6 / 8×8 / 10×10、デフォルト 8×8）
- [x] 多人数リバーシロジック（全員挟み判定・反転）
- [x] 色覚バリアフリー対応駒（Okabe-Itoパレット: 青・橙・紫・緑 + 模様で識別）
- [x] 公平な初期配置（中央集中・ランダム回転）
- [x] パス制ターン管理・終局判定
- [x] プレイヤーパネル・スコア表示
- [x] 人数選択画面
- [x] 盤面サイズ選択機能
- [x] 色の順番ドラッグ並べ替え（ゲーム開始前にプレイヤーの色順をドラッグで変更可能）
