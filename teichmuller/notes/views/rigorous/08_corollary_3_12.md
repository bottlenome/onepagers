# 系3.12の争点 — 学会が受容していない核心

**論文参照**: IUT III, Corollary 3.12
**争点の発見**: Peter Scholze & Jakob Stix (2018)
**この文書の目的**: 「数学界は具体的に何を問題視しているか」を厳密に記述する

---

## 系3.12が主張すること

### 位置づけ

系3.12は IUT III の §3 の最後に位置し、理論全体の核心的な見積もりを与える。
この系が正しければ、IUT IV での計算を経てABC予想が従う。
この系に欠陥があれば、理論全体の帰結が崩壊する。

### 主張の要約

対数テータ格子上のマルチラジアルアルゴリズムにより、
LGP-モノイドの分裂モノイドの対数体積について**非自明な上界**が得られる。

具体的には、三つの不定性 (Ind1), (Ind2), (Ind3) を考慮しても、
テータ値 {q^{j²}} の対数体積の見積もりが q-パラメータの対数体積に対して
非自明な（つまり情報を含む）関係を与える。

## Scholze-Stix の批判の核心

### 2018年の訪問

Scholze（フィールズ賞2018受賞、ボン大学）と Stix（フランクフルト大学）は
2018年3月に京都の RIMS を訪問し、望月と1週間にわたって議論した。

### 批判の骨子

Scholze-Stix の主張を正確に述べると:

**「系3.12の証明において、Θ-リンクを跨いだ情報の受け渡しを追跡すると、
三つの不定性（特にInd1とInd2）が結合した効果により、
得られる不等式は自明（trivial）になる。つまり、新しい情報は何も得られない。」**

### 技術的な争点

#### 問題の所在: Θ-リンクと局所データの同一視

1. Θ-リンクは二つのホッジシアター †HT と ‡HT を繋ぐ
2. これは環準同型ではない → 足し算の構造が両側で異なる
3. 系3.12の証明では、†HT 側のテータ値の情報を ‡HT 側の q-パラメータと比較する
4. この比較の過程で、遠アーベル幾何学的復元を使って「異質な算術的正則構造」を橋渡しする

#### Scholze の具体的指摘

Scholze は、望月の議論を以下のように「簡略化」した（と主張する）:

```
望月の議論:
  †HT の環構造 →(遠アーベル復元)→ 抽象的データ →(マルチラジアル)→ ‡HT の環構造

Scholze の簡略化:
  この過程で、Ind1（内部自己同型の不定性）が自由に作用できるため、
  †HT のテータ値と ‡HT の q-パラメータの間の関係が
  「任意の値を取りうる」ことになり、不等式が自明化する
```

#### Scholze の主張を図示

```
†HT側:  テータ値 {q^{j²}}
    ↓ Θ-リンク（環構造の非互換性）
    ↓ 遠アーベル復元（Ind1の不定性あり）
    ↓
‡HT側:  q-パラメータ

Scholze: 「↓の過程でInd1の不定性が全体を飲み込み、
         出力は入力に依存しない（= 自明）」
```

## 望月の反論

### 反論の骨子

望月は Scholze-Stix の批判に対して詳細な反論文書を公開した。主張:

**「Scholze-Stix の『簡略化』は、理論の本質的な構造を取り除いている。
特に、対数テータ格子の縦方向（log-リンク）の反復により構成される
対数殻の理論を無視している。」**

### 技術的な反論

1. **対数殻による制約**: Ind1 の不定性は無制限に作用するのではなく、
   対数殻 I_v の内部に制限される。対数殻は log-リンクの反復で構成され、
   明確なサイズ（対数体積）を持つ。

2. **マルチラジアリティの誤解**: Scholze-Stix の「簡略化」は
   ユニラジアルな視点からの分析であり、マルチラジアルな構造を正しく捉えていない。

3. **log-Kummer対応**: 対数テータ格子の縦方向（log-リンク）と
   Kummer理論の組み合わせ（log-Kummer対応）が、
   不定性を対数殻の範囲内に制限する役割を果たす。

### 望月の反論を図示

```
†HT側:  テータ値 {q^{j²}}
    ↓ Θ-リンク（環構造の非互換性）
    ↓ log-リンクの反復（対数殻の構成）    ← Scholzeが無視していると主張
    ↓ 遠アーベル復元（Ind1の不定性あり）
    ↓ ただし対数殻 I_v の範囲内に制限    ← これが非自明な見積もりの源泉
    ↓
‡HT側:  q-パラメータ

望月: 「対数殻が不定性を制限するので、不等式は非自明」
```

## Scholze の再反論 (2021)

Scholze は2021年に再度、以下の趣旨で反論:

- 望月の「対数殻による制約」の議論を検討したが、依然として説得されない
- 対数殻が Ind1 の不定性を本当に制限するかどうかについて、
  望月の議論は十分に明確でない
- 結論: 系3.12の証明には依然としてギャップがある

## Joshi の見解 (2024–2025)

Kirti Joshi（アリゾナ大学）は第三の立場を提示:

### Joshi の主張

1. **Scholze の2021年の批判は「数学的に誤り」**:
   Scholze が構成した「反例」は、IUT理論の前提条件を満たしていないと主張

2. **しかし望月の証明も不完全**:
   遠アーベル幾何学的手法（群論的手法）だけでは、
   マルチラジアルな文脈での復元を正当化するのに不十分

3. **「幾何的対象の優位性」**:
   望月は群（ガロア群、基本群）を重視しすぎており、
   幾何的・算術的対象そのものの構造をより直接的に扱うべき

4. **Joshi 自身のアプローチ**:
   p進幾何（ダイアモンド、プリズム等の近年の道具）を用いた再構成を試みている

### 2024年6月: Scholze との対話

Joshi によれば、Scholze と以下の点で合意:
- 望月の手法は「群論的に純粋すぎる」
- 幾何的・算術的対象の primacy（優位性）が重要

ただし、Joshi の新しい証明が正しいかどうかは Scholze を含め未検証。

## 争点の構造化まとめ

### 三者の立場

| | 系3.12は正しいか | 不等式は非自明か | 何が不足しているか |
|---|---|---|---|
| **望月** | 正しい | 非自明 | 何も不足していない |
| **Scholze** | 正しくない | 自明化する | Ind1の制御が不十分 |
| **Joshi** | 証明は不完全 | （正しい手法で）非自明にできるはず | 群論的手法を幾何的手法に置き換える必要 |

### 対立の核心を一言で

> **「遠アーベル幾何学的復元のInd1不定性は、対数殻によって十分に制約されるか？」**

- 望月: Yes → 系3.12は正しい → ABC予想が証明される
- Scholze: No → 系3.12は成り立たない → 証明は無効
- Joshi: 望月の方法ではNo、しかし別の方法ならYesの可能性

1. 望月は自身の証明を修正する意図がない（欠陥を認めていない）
2. Scholze は批判を撤回する意図がない（説得されていない）
3. Joshi の再構成はまだ完成・検証されていない
4. 他の数学者の多くは、理論全体の理解に到達していないため判断を保留

数学界の暗黙の合意: **ABC予想は未解決問題として扱われている**。
