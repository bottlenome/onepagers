# マルチラジアルアルゴリズム [IUT II §1–§2, IUT III §2–§3]

---

## 定義

### 算術的正則構造 (Arithmetic Holomorphic Structure)

ホッジシアターに付随する環/スキーム構造のこと。
古典的複素解析の「正則構造」（複素構造）の算術版。

各ホッジシアター n,m HT は固有の算術的正則構造を持つ。
Θ-リンクで繋がれた二つのホッジシアターは、異なる算術的正則構造を持つ。

### ユニラジアル (Uniradial)

一つの算術的正則構造の中でのみ定式化できる概念・アルゴリズム。
通常の環論・スキーム論の議論はすべてユニラジアル。

### マルチラジアル (Multiradial)

異なる算術的正則構造の視点からも意味をなす概念・アルゴリズム。

**厳密な定義** [IUT II, Introduction]:
アルゴリズム A が「マルチラジアル」であるとは、
Θ-リンクの両側のホッジシアターから同じ出力を与えること。

すなわち、†HT と ‡HT が Θ-リンクで繋がれているとき:

```
A(†HT 側のデータ) = A(‡HT 側のデータ)
  （適切な意味での等号）
```

## マルチラジアリティの実現手段

### エタールテータ関数の剛性 [EtTh]

マルチラジアリティを実現するための中心的な道具:

| 剛性 | 出典 | 内容 |
|------|------|------|
| **単テータ環境剛性** | [EtTh, Cor 2.19] | mono-theta environment がエタール基本群から一意に復元できる |
| **離散剛性** | [EtTh, Prop 1.4] | テータ関数の冪の不定性が Z のみ（連続的な不定性がない）|
| **定数倍剛性** | [EtTh, Cor 2.19] | テータ関数の定数倍の不定性が μ_{2l}（有限群）のみ |

### 共役同期 (Conjugate Synchronization) [IUT II, §2]

l-等分点 j = 0, ±1, ..., ±l* のそれぞれに付随する局所絶対ガロア群の
コピーが**共役の不定性なしに**同一視されるシステム。

技術的には:
- F_l^± 対称性（加法的対称性）がこの同期を実現
- IUT I で解決された「テンパード尖点惰性群のプロ有限共役」の結果を使用

## 三つの不定性の厳密な記述

### (Ind1): Aut_G-不定性

**由来**: 遠アーベル幾何学的復元における不定性

エタール基本群 Π_X からスキーム X を復元するとき、
Π_X の内部自己同型 (inner automorphism) による不定性が残る。

```
X →(基本群関手)→ Π_X →(復元関手)→ X'
                       ↑
              Inn(Π_X) が作用 ← これが Ind1
```

**影響**: 復元されたオブジェクトが「内部自己同型の軌道」として定まる。

### (Ind2): O^×-不定性

**由来**: 対数殻上の単数群の作用

対数殻 I_v 上で O_{K_v}^×（局所単数群）が掛け算で作用する。
この作用を法として対数体積を見積もる。

```
I_v 上のデータ → O_{K_v}^× · I_v = I_v（対数殻は O^× 作用で不変）
```

**影響**: テータ値の「絶対的な値」は定まらないが、対数殻の範囲は定まる。

### (Ind3): 上限/下限の不定性

**由来**: 対数体積の見積もりにおける評価の幅

対数殻の対数体積は「上からの評価」と「下からの評価」の間に幅がある。

**影響**: 最終的な不等式に「定数項」として反映される。

## マルチラジアル表現の核心

### 何がマルチラジアルに表現できるか

| データ | マルチラジアルか | 理由 |
|--------|----------------|------|
| D-データ（エタール基本群的データ）| Yes | Θ-リンクで保存される |
| フロベニオイド的データ | No（一般には）| Θ-リンクで変わる |
| テータ値 q^{j²}（対数殻内で） | Yes（Ind1,2,3 を込めて）| 剛性定理による |
| 環構造そのもの | No | Θ-リンクで壊される |
| 対数殻 I_v | Yes | log-リンク反復で不変 |

### 系3.12 への帰結

マルチラジアルに表現できるデータ（テータ値の対数殻内での像）と、
Θ-リンクの反対側の q-パラメータを比較することで、
**三つの不定性を込めた対数体積の不等式**が得られる（と望月は主張）。
