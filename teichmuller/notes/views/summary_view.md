# 概要ビュー — IUT理論を3段階で理解する

このビューでは「で、結局何なの？」に簡潔に答える。

---

## 30秒で理解する

**IUT理論** = 望月新一が約1000ページかけて構築した数学理論。
**目標**: 整数論の超重要未解決問題「ABC予想」を証明すること。
**方法**: 数の世界の「足し算」と「掛け算」を一度バラバラにして組み直す。

---

## 3分で理解する

### 1. 問題: ABC予想

「a + b = c を満たす互いに素な整数について、c の大きさは素因数の積 rad(abc) でほぼ抑えられる」

→ 足し算の結果が掛け算の構造で縛られるという、驚くべき主張。

### 2. 難しさ: 足し算と掛け算の不可分性

普通の数学では、足し算と掛け算は一つの「環」として一体化している。
片方だけを動かして不等式を出すことができない。
既存の手法（スキーム論、代数幾何学）はすべてこの一体構造の上に立っている。

### 3. 解決: 環構造の解体と再構成

望月のアイデア:
1. 環構造を「足し算的パーツ」と「掛け算的パーツ」に解体（ホッジシアター）
2. 掛け算側だけを変形する橋を架ける（シータリンク: q^{j²} ↔ q）
3. 遠アーベル幾何学で、変形後の世界から元の環構造を復元する
4. 復元の際の「歪み」を定量化して不等式を導く（ABC予想の形）
