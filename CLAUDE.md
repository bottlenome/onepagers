# CLAUDE.md

このリポジトリでの開発ルール。

## ダッシュボード更新ルール

プロジェクトのコードを変更した場合、関連するダッシュボードも必ず更新すること。

1. **プロジェクト個別の `dashboard.md`** — 機能追加・変更・バグ修正時に該当プロジェクトの `<project>/dashboard.md` を更新する
2. **ルートの `dashboard.md`** — プロジェクトの追加・削除・説明変更時にプロジェクト一覧テーブルを更新する
3. **ルートの `index.html`** — プロジェクトの追加・削除・説明変更時にカードの内容を更新する

### チェックポイント

- 新規プロジェクト追加時: `<project>/index.html`, `<project>/dashboard.md`, ルート `dashboard.md`, ルート `index.html` の4箇所を更新
- 既存プロジェクト変更時: `<project>/dashboard.md` の機能リストや説明が現状と一致しているか確認
- 説明文の変更時: ルート `dashboard.md` のテーブルとルート `index.html` のカードの両方を同期
