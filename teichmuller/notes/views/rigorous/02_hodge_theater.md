# ホッジシアターの構成 [IUT I, §3–§6]

---

## Θ±ell NF-ホッジシアターの定義の骨格

ホッジシアターは複数の圏論的データの組として定義される。
完全な定義は IUT I の §3–§6 に跨る約200ページの構成だが、
ここでは構造の骨格を記述する。

### 構成要素

```
Θ±ell NF-ホッジシアター = D-Θ±ell NF-HT ⊕ Frobenioid的データ
```

#### D-Θ±ell NF-ホッジシアター

基底にある圏論的データ。「D」は base category (D = Dias, 基底圏) に由来。

| 成分 | 定義の出典 | 役割 |
|------|-----------|------|
| D-Θ±ell-HT | IUT I, Def 6.4 | 加法的 (F_l^±) 対称性を記述 |
| D-ΘNF-HT | IUT I, Def 5.5 | 乗法的 (F_l^⊛) 対称性を記述 |

#### 加法的対称性 F_l^±

- F_l = Z/lZ に ±1 の作用を加えた群: F_l^± = F_l ⋊ {±1}
- 作用: l-等分点のラベル {0, ±1, ±2, ..., ±l*} に対する対称性
- ラベル 0 は「ε」（初期シータデータのカスプ）に対応

#### 乗法的対称性 F_l^⊛

- F_l^⊛ = F_l^× / {±1}（零元を除いた F_l の乗法群を ±1 で割ったもの）
- 作用: 値群的な構造に対する対称性
- NF（Number Field）のデータ（数体の大域的情報）と結びつく

### 二つの対称性の「分離」

**重要**: 通常の環 F_l では加法と乗法は分配法則で結合している。
ホッジシアターでは、この結合を**意図的に切断**し、
加法的パーツと乗法的パーツを別々のデータとして保持する。

## フロベニオイド的データ

各ホッジシアターにはフロベニオイド（望月が開発した圏論的構造）が付随する:

| フロベニオイド | 対応する数論的データ |
|--------------|-------------------|
| Θ-フロベニオイド | テータ関数に関わるモノイド |
| NF-フロベニオイド | 数体の整数環に関わるモノイド |

フロベニオイドは「環」から掛け算的構造だけを抽出した圏論的対象。

---

## ホッジシアター間の同型

**命題**: すべての Θ±ell NF-ホッジシアターは互いに同型である。

つまり、異なるホッジシアターは「構造として区別できない」。
しかし、シータリンクで繋ぐとこの同型は**環構造と両立しない**。
この「同型だが環構造が異なる」状況がIUT理論の核心。
