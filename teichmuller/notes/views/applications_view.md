# 何ができるかビュー — IUT理論が証明する（と主張する）もの

このビューでは「この理論で結局何が証明できるの？」に答える。

**注意**: IUT理論の第一義的な目標はABC予想の証明ではなく、
「数体の算術的構造の変形理論」の構築である。
ABC予想はその理論から導かれる最も顕著な帰結（応用）。

---

## 理論そのものの成果（構成的成果）

IUT理論が構築した概念的枠組み自体が成果:

| 成果 | 内容 |
|------|------|
| **環構造の解体と再構成** | 数体の足し算と掛け算を分離して扱う枠組みの提示 |
| **対数テータ格子** | シータリンクとlog-リンクからなる二次元非可換ダイアグラム |
| **マルチラジアリティ** | 異なる環構造をまたいで意味をなすアルゴリズムの概念 |
| **Speciesの理論** | 集合論モデルを跨ぐ数学的対象の型の形式化 |

これらは**証明の正否に関わらず**、新しい視点として提示されたもの。

## ディオファンタス幾何への帰結（証明が正しいと仮定した場合）

### 1. Vojta予想（双曲曲線の場合）— IUT IV 定理Aの直接的な内容

**主張**: IUT IV の定理A。双曲的曲線上の代数的点の高さの上界。

```
ht_{ω_X(D)} ≤ (1 + ε)(log-diff_X + log-cond_D)
```

**意味**: 双曲曲線上の有理点の「大きさ」は、定義体と悪い還元の情報で制約される。
これが理論の最も一般的な帰結であり、以下の予想はすべてこの特殊ケース。

### 2. ABC予想（Vojta予想の特殊ケース）

**主張**: 互いに素な正整数 a + b = c に対し、任意の ε > 0 で

```
c < K(ε) · rad(abc)^{1+ε}
```

が成り立つ（K(ε) は ε のみに依存する定数）。

**意味**: 足し算の結果は、関与する素因数の種類で本質的に制約される。
Vojta予想で X = P¹, D = {0, 1, ∞} とした特殊ケース。

### 3. Szpiro予想

**主張**: 楕円曲線 E/Q の最小判別式 Δ と導手 N に対し

```
|Δ| ≤ C(ε) · N^{6+ε}
```

**意味**: 楕円曲線の「複雑さ」（判別式）は、「どこで悪い還元を持つか」（導手）で制約される。

### (参考) Vojta予想の再掲

上記の1.で述べた通り、Vojta予想は IUT IV 定理Aの直接的な内容。
ABC予想とSzpiro予想はその特殊ケース。

## 間接的な帰結（ABC予想から従うもの）

| 帰結 | 内容 | ABC予想との関係 |
|------|------|----------------|
| **フェルマーの最終定理（有効版）** | x^n + y^n = z^n は n ≥ 4 で非自明解なし。しかも具体的な上界が得られる | ABC → 有効的FLT |
| **Pillai予想の部分結果** | 完全冪の差 |x^a - y^b| の下界 | ABC → Pillai型評価 |
| **Erdős–Woods予想の部分結果** | 連続する整数列の素因数による特徴づけ | ABC → 部分的解決 |
| **弱いHall予想** | x³ - y² ≠ 0 ならば |x³ - y²| > |x|^{1/2-ε} | ABC → Hall |
| **Lang予想の特殊ケース** | 一般型多様体上の有理点の有限性の定量版 | Vojta → Lang |

## IUT理論固有の副産物

ABC予想の証明以外にも、IUT理論の構築過程で生まれた概念や技術がある:

### 遠アーベル幾何学の発展
- エタール基本群からスキームを復元するアルゴリズムの精密化
- 双曲曲線の遠アーベル的剛性の新しい結果

### エタールテータ関数の理論 [EtTh]
- テータ関数のエタール的（代数的）な取り扱い
- 3種の剛性（単テータ環境剛性・離散剛性・定数倍剛性）

### フロベニオイドの理論 [FrdI], [FrdII]
- 数論的対象の掛け算的構造を圏論的に抽象化する新しい枠組み

### Speciesの理論（集合論的基礎）
- 「数学的対象の型」を集合論的公式で形式化
- 複数の集合論モデルを跨ぐ操作の正当化

## 「できないこと」の明確化

| できないこと | 理由 |
|-------------|------|
| 高次元アーベル多様体への一般化 | IUT理論は楕円曲線（1次元）に本質的に依存。ガウス積分の高次元化が自明でないのと同じ |
| 具体的な定数の計算 | 不等式は「bounded discrepancy class」の意味。定数は非構成的 |
| ABC予想の最適な指数の決定 | (1+ε) という形でのみ成立。ε=0 のケース（強い形のABC）は射程外 |
| リーマン予想への応用 | 直接的な関連はない（ただし遠い将来の可能性は否定されない）|
