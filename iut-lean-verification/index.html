結論（形式的判定）

IUT の正否は「Θ-リンクの両側のコピーを同一視してよいか」という一点に形式的に帰着する。これが Lean で機械検証された判定であり、二分法として厳密に証明された（定理 IUT.verdict）:

  1. Scholze–Stix の読み（同一視を認める）なら、系3.12 は証明可能な形で矛盾する。すなわちこの読みのもとで「IUT は間違っている」は Lean 検証済みの定理である（ss_incompatible）。
  2. 望月の読み（同一視を拒否する）なら、矛盾は導出されない。骨格と系3.12 は無矛盾であり、Scholze–Stix 型の反証はこの読みでは再現不能（cor312_consistent）。
  3. ただし同じ読みのもとで、系3.12 は形式骨格から導出もできない（独立命題）。「IUT が正しい」ことの全根拠は、未形式化の定理3.11（多輻的アルゴリズム）の実質的内容に残る（cor312_not_derivable）。

つまり「IUT の完全な正しさ」も「完全な間違い」も無条件には証明されない。形式的に証明できる最強の主張は「論争は RC 同一視の採否と外延的に等価」（controversy_reduces_to_rc）であり、本プロジェクトはそれを機械検証した。

−|log(Θ)| ≥ −|log(q)|   （ただし |log(q)| > 0）
  • |log(q)| — q-パイロット対象（楕円曲線の q-パラメータ）の procession 正規化対数体積
  • |log(Θ)| — Θ-パイロット対象（テータ値 q1², …, ql⋇² の集まり、l⋇ = (l−1)/2）の、不定性 (Ind1)(Ind2)(Ind3) 込みの「可能な像の合併の正則包」の対数体積

素朴に計算すると Θ 側の平均次数は (Σ j²)/l⋇ > 1 倍なので、不等式は「大きいはずのものが小さい側に収まる」と主張している。この一見した矛盾をどう解釈するかが2018年以降の論争（Scholze–Stix vs 望月）の核心である。

形式化の方法

1,000ページ全体の形式化は現実的に不可能であり、また論争の係争点はそこにはない。本プロジェクトは両陣営が合意する数値データだけを「形式骨格」として公理化し、係争点である「コピーの同一視」を分離した追加公理 RCEval として定式化した。

structure Skeleton where
  lstar    : Nat        -- l⋇ = (l−1)/2（テータ値のラベル数）
  hl       : lstar ≥ 2   -- l ≥ 5 より
  logq     : Int        -- |log(q)|
  hq       : logq > 0    -- 系3.12 の仮定
  logTheta : Int        -- |log(Θ)|（不定性込みの多輻的体積）

/-- 系3.12 の主張 -/
def Cor312 (s : Skeleton) : Prop := -s.logTheta ≥ -s.logq

/-- RC 同一視（Scholze–Stix）のもとで成立する評価:
    l⋇·|log(Θ)| ≥ (Σ j²)·|log(q)| -/
def RCEval (s : Skeleton) : Prop :=
  (s.lstar : Int) * s.logTheta ≥ (sumSq s.lstar : Int) * s.logq

準備として、テータ値の平均次数が 1 を真に超えること（Σj=1l⋇ j² > l⋇、l⋇ ≥ 2）を帰納法で証明した（sumSq_gt、Lean core のみ使用）。

検証された3定理

定理1: Scholze–Stix の退化（同一視 → 矛盾）

theorem ss_incompatible (s : Skeleton)
    (hrc : RCEval s) (hcor : Cor312 s) : False

系3.12 から |log(Θ)| ≤ |log(q)|、RC 評価から l⋇·|log(Θ)| ≥ (Σj²)·|log(q)|。連結すると (Σj²)·|log(q)| ≤ l⋇·|log(q)|、しかし Σj² > l⋇ かつ |log(q)| > 0 なので矛盾。Scholze–Stix の「同一視すれば不等式は破綻する」という2018年の指摘は、数学的に厳密であることが機械検証された。

定理2: 望月の読みの無矛盾性（同一視なし → 矛盾は消える）

theorem cor312_consistent : ∃ s : Skeleton, Cor312 s

RC 評価を公理から外すと、系3.12 が成り立つモデルが存在する。望月の「コピーは別個の対象であり、同一視こそが誤解の根源」という反論（Essential Logical Structure of IUT, 2024）にも形式的な実体がある——同一視を拒否する限り、Scholze–Stix 型の反証は構成できない。

定理3: 系3.12 の独立性（同一視なし → 証明もできない）

theorem cor312_independent :
    (∃ s : Skeleton, Cor312 s) ∧ (∃ s : Skeleton, ¬Cor312 s)

同時に、系3.12 が成り立たないモデルも存在する。すなわち系3.12 は形式骨格から独立であり、その成立は形式化の外側——定理3.11 の多輻的アルゴリズムが本当に主張通りの体積制御を与えるか——に全面的に依存する。これが論争の真の係争点であり、未決着のまま残る部分である。

付録: 望月の ∧/∨̇ 論理構造の検証

「Essential Logical Structure of IUT」(2024) で望月は IUT の論理構造を次の命題論理式に要約している:

A ∧ B = A ∧ (B₁ ∨̇ B₂ ∨̇ …) ⟹ A ∧ (B₁ ∨̇ … ∨̇ B′₁ ∨̇ …)

∨̇ は排他的論理和 XOR（不定性 Ind1–Ind3 による分岐）。この表示自体を命題論理として検証した結果（IUT/Boolean.lean）:

検証項目結果
包含的 OR への弱化 A∧B → A∧(B∨C)✅ 無条件に妥当（and_or_weakening）
XOR 連鎖への弱化❌ 一般には不成立。反例を構成（xor_weakening_fails）
追加選択肢が偽と判明している場合の XOR 弱化✅ 妥当（xor_weakening_of_not, exactlyOne_extend）
RC 同一視による A∧B → A∧A の退化✅ A∧A ↔ A、∧ の論理的内容が消失（and_self_collapse）

つまり望月の表示を文字通り XOR と読むなら、選択肢を追加する弱化ステップには「新選択肢が既存選択肢と両立しない」ことの証明が別途必要で、それは命題論理の外（定理3.11 の内容）に属する。ここでも結論は同じ係争点に帰着する。

本検証の限界

  • 形式化したのは論争の算術的骨格であり、定理3.11 の多輻的アルゴリズムの内部（遠アーベル幾何による復元、log-Kummer 対応など）は形式化していない。これらの完全形式化は現時点で誰にも達成されていない。
  • 対数体積は実数ではなく整数（共通分母で正規化）でモデル化した。議論はすべて順序環の不等式論法なので結論に影響しない。
  • 「RC 同一視が数学的に許されるか」自体は形式体系の外の問い（どちらを公理に取るかの選択）であり、本検証はその選択がもたらす帰結を完全に分類したものである。
