## 論理的に最も脆弱なリンク

### 単一障害点 (Single Point of Failure)

```
[IUT I–II の構成] → [IUT III 系3.12] → [IUT IV の帰結]
                          ↑
                    ここが崩れると
                    すべての帰結が失われる
```

系3.12は証明全体の「ボトルネック」:
- この系が正しければ、IUT IV の計算は標準的な見積もりの組み合わせで完結する
- この系に欠陥があれば、ABC予想の証明は成立しない

### 争いのない部分

| 部分 | 状態 |
|------|------|
| IUT I: ホッジシアターの構成 | 技術的に正しいと見なされている |
| IUT II: テータ評価の基礎理論 | 概ね受け入れられている |
| IUT IV §1: 対数体積の計算 | 標準的な見積もり。系3.12が正しければ問題なし |
| IUT IV §2: GenEll による帰着 | [GenEll] 自体は独立に検証されている |
| IUT IV §3: Species の理論 | 数学的帰結とは独立。基礎論的に興味深いが論争の対象外 |

### 争いのある部分

|------|------|
