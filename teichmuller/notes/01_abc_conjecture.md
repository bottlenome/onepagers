# ABC予想とは何か

## 一言でいうと

「足し算の結果と、素因数の構造には、深い関係がある」という予想。

## 正式な定義

3つの正の整数 a, b, c が以下を満たすとする:
- a + b = c
- a, b, c は互いに素（共通の因数を持たない）

このとき **radical（根基）** rad(abc) を「abc の異なる素因数の積」と定義する。

**ABC予想**: 任意の ε > 0 に対して、有限個の例外を除き、以下が成り立つ:

```
c < rad(abc)^(1+ε)
```

## 具体例で理解する

### 例1: a=1, b=8, c=9
- 1 + 8 = 9
- rad(1 × 8 × 9) = rad(1 × 2³ × 3²) = 1 × 2 × 3 = 6
- c=9 > rad=6 → c が radical を超えている！（こういうケースは稀）

### 例2: a=5, b=27, c=32
- 5 + 27 = 32
- rad(5 × 27 × 32) = rad(5 × 3³ × 2⁵) = 5 × 3 × 2 = 30
- c=32 > rad=30 → ギリギリ超えている

### 直感的な理解

素因数が「小さい種類の素数の高い冪乗」で構成されている数同士を足して、また「小さい種類の素数の高い冪乗」になるケースは**非常に稀**。ほとんどの場合、足し算をすると素因数の構造がバラバラになる。

## なぜ重要か

ABC予想が真であれば、以下が全て従う:

| 帰結 | 内容 |
|------|------|
| **フェルマーの最終定理**（大きな指数）| x^n + y^n = z^n (n≥4) の非自明な整数解なし |
| **Szpiro予想** | 楕円曲線の判別式と導手の関係 |
| **Vojta予想**（双曲線の場合）| ディオファンタス幾何の中心的予想 |
| **Roth型の改良定理** | 代数的数の有理近似に関する精密化 |

つまり、ABC予想は整数論の「メタ定理」であり、証明できれば膨大な数の結果が一気に得られる。

## 足し算 vs 掛け算の対立

ABC予想の本質は「足し算の構造」と「掛け算の構造（素因数分解）」の間の緊張関係にある。

- **掛け算の世界**: 数を素因数の積として理解する → 120 = 2³ × 3 × 5
- **足し算の世界**: 数を和として理解する → 120 = 50 + 70

この2つの操作は独立のように見えて、実は深い拘束関係にある。それを定量的に述べたのがABC予想。
