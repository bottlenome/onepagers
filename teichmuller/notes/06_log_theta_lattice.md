## 対数テータ格子 (Log-theta-lattice)

### 二次元の非可換ダイアグラム

```
        ...                 ...                 ...
         ↑ log               ↑ log               ↑ log
  ... ← •_{n,m+1} ← Θ ← •_{n+1,m+1} ← Θ ← •_{n+2,m+1} ← ...
         ↑ log               ↑ log               ↑ log
  ... ← •_{n,m}   ← Θ ← •_{n+1,m}   ← Θ ← •_{n+2,m}   ← ...
         ↑ log               ↑ log               ↑ log
  ... ← •_{n,m-1} ← Θ ← •_{n+1,m-1} ← Θ ← •_{n+2,m-1} ← ...
         ↑ log               ↑ log               ↑ log
        ...                 ...                 ...
```

- 各 **•** はホッジシアター（環構造のミニチュアモデル）
- **横矢印 (Θ)**: シータリンク（掛け算的な変形、環構造を壊す）
- **縦矢印 (log)**: 対数リンク（p進対数写像の適用）
- **非可換**: 「右→上」と「上→右」で結果が異なる！

### なぜ「格子」なのか

整数格子 Z² のように、横と縦の二方向に無限に広がるダイアグラムだから。
ただし格子の矢印は可換ではない（順番が重要）。

---

## 対数リンク (Log-link)

### 縦方向の矢印

各非アルキメデス付値 v で、局所 p_v 進対数写像を適用する操作:

```
局所単数 O_v^× →(log)→ p進対数の像
```

### 対数殻 (Log-shell)

対数リンクの像を少し調整した集合。

**直感**: 掛け算的な構造（単数群 O_v^×）を、対数を通じて足し算的な構造に変換し、
その結果得られる「殻」（入れ物）。

対数殻は「足し算と掛け算を橋渡しする入れ物」として機能する。

## マルチラジアルアルゴリズムの完成

### IUT IIで導入された概念の本格展開

対数テータ格子上で、あるホッジシアター（格子点）の算術的正則構造とは
**異なる**算術的正則構造（別の格子点）の視点からも意味をなすアルゴリズムを構築。

### LGP-モノイド

**Logarithmic Gaussian Procession Monoid**（対数的ガウス行列モノイド）

- IUT IIの「ガウスモノイド」の対数殻版
- テータ値の情報を対数殻の言葉に翻訳したもの
- マルチラジアルアルゴリズムによって「分裂モノイド」が構成できる

### 分裂 (Splitting)

LGP-モノイドが「正しく分裂する」ことが確認される:
- これは、テータ値の情報が対数テータ格子の操作を通じても保存されることを意味する
- この分裂から対数体積の見積もりが可能になる

## 三つの不定性 (Indeterminacies)

マルチラジアルアルゴリズムの適用時に避けられない3種類の不定性:

| 不定性 | 内容 | 由来 |
|--------|------|------|
| **(Ind1)** | ガロア群の内部自己同型 | 遠アーベル幾何の復元における不定性 |
| **(Ind2)** | 単数群 O^× の作用 | 対数殻上の乗法的不定性 |
| **(Ind3)** | 対数殻の上限・下限の選択 | 対数体積の評価における不定性 |

これらの不定性は「歪み」として最終的な不等式に反映されるが、
十分に制御可能であり、ABC予想の形の不等式を導くのに支障はない（と主張される）。

## Essential Logical Structure との対応

2024年の論文「Essential Logical Structure of IUT」では、この構造を命題論理で表現:

```
A ∧ B ⟹ A ∧ (B₁ ∨̇ B₂ ∨̇ ...)
```

- **∧ (AND)**: シータリンク（横方向）→ 掛け算的構造の対応
- **∨̇ (XOR)**: 不定性（主に対数リンクから生じる）→ 足し算的構造の「揺らぎ」

AND と XOR の連鎖は、ヴィット環の繰り上がり加算の構造と類似する。
