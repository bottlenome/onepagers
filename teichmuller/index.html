宇宙際タイヒミューラー理論 (IUT)

望月新一が約1000ページかけて構築した数学理論。整数論の超重要未解決問題「ABC予想」を、数の世界の足し算と掛け算を一度バラバラにして組み直すことで証明しようと試みる。

数論幾何 ABC予想 遠アーベル幾何 タイヒミューラー理論 非数学者向け解説

30秒で理解する

IUT理論は、整数論の超重要未解決問題「ABC予想」を証明するために、 数の世界の足し算と掛け算を一度バラバラに解体し、別々の"宇宙"に分けて、 繋ぎ直すことで不等式を導く理論。

著者は望月新一（京都大学RIMS）。論文4本＋概観2本で合計約1,000ページ超。 2012年に公開、2021年にPRIMSに掲載されたが、Fields賞受賞者Scholzeを含む数学コミュニティの 多数派は「ABC予想はまだ未解決」と見なしている（2025年時点）。

3分で理解する

1. 問題

ABC予想

「a + b = c を満たす互いに素な整数について、c の大きさは素因数の積 rad(abc) でほぼ抑えられる」。足し算の結果が掛け算の構造で縛られるという驚くべき主張。

2. 難しさ

足し算と掛け算の不可分性

普通の数学では足し算と掛け算は一つの「環」として一体化していて、片方だけを動かして不等式を出すことができない。スキーム論など既存の手法はすべてこの一体構造の上に立つ。

3. 解決

環構造の解体と再構成

環構造をホッジシアターに解体 → 掛け算側だけシータリンクで変形 (q^j² ↔ q) → 遠アーベル幾何で復元 → 復元の歪みを定量化 → ABC予想の不等式。

「なぜ？」を10問で理解する

各段階で「なぜそうするのか？」「なぜそれが必要なのか？」に答える。

核心の問い

Q1: なぜABC予想は直接証明できないのか？

A: 足し算と掛け算が同じ環の中で不可分に結びついているから。

  • 既存の数学的枠組み（スキーム論、代数幾何）はすべて環構造を前提にしている
  • 環の中では足し算と掛け算は分配法則で絡み合っている: a(b+c) = ab + ac
  • 片方だけを「動かす」操作が、そもそも定義できない

具体例: 5進数の世界で 7 + 18 = 25 を考える。7 も 18 も「5とは無関係」（v₅=0, 単数）なのに、足すと 25=5² で「5が2つ出てくる」。足し算が掛け算の構造を変えてしまう。

Q2: なぜ環構造を「壊す」のか？

A: 壊さないと足し算と掛け算を独立に扱えないから。

望月は2000年にホッジ・アラケロフ理論を開発し、環構造を保ったまま不等式を導こうとした。しかし「ガウス極」という技術的障害により、得られる不等式が空虚 (vacuous) になった。これが「環構造を壊す」という大胆なアプローチへの動機となった。

Q3: なぜホッジシアターという複雑な構造が必要なのか？

A: 環構造を壊すと通常の数学（スキーム論）が使えなくなるので、代替物が必要だから。

ホッジシアターはスキーム論のミニチュアモデル。環構造の代わりに、足し算的対称性 (F_l±) と掛け算的対称性 (F_l⊛) を分離して保持する。これにより、環を壊した後も「何が何に対応するか」を追跡できる。

Q4: なぜ素数 l が必要なのか？

A: 有限体 F_l = Z/lZ が足し算と掛け算の両方の対称性を持つ最小の舞台だから。

  • F_l は加法群と乗法群 F_l* の両方を持つ
  • l が素数であることで、これらの対称性が「きれいに」分離できる
  • l ≥ 5 は技術的な理由（l=2,3 では退化が起きる）

最終的に l は楕円曲線の高さと同じオーダー（l ~ ht_E）に選ばれる。

Q5: なぜシータ関数なのか？

A: テータ関数 θ(q) = Σ qn² が「ガウス分布の離散版」として、足し算と掛け算を橋渡しする唯一の自然な対象だから。

  • q^j² の形: 二次形式 (j²) がガウス分布 e−x² の離散版
  • 楕円曲線との関係: テータ関数は楕円曲線のモジュライ上で自然に定義される
  • 剛性: エタールテータ関数は離散剛性・定数倍剛性・単テータ環境剛性を持ち、マルチラジアルな文脈でも well-defined
Q6: なぜ対数 (log-link) が必要なのか？

A: p進対数は「掛け算を足し算に変換する」唯一の自然な操作だから。

掛け算的な対象（局所単数 O_v×）を足し算的な対象（対数殻）に変換する。この変換により、シータリンク（横）と対数リンク（縦）が二次元の格子を形成し、足し算と掛け算を「別の方向」として扱うことが可能になる。

Q7: なぜマルチラジアリティが必要なのか？

A: シータリンクは環構造を壊すので、「こちら側」の言葉が「あちら側」で意味をなさなくなるから。

  • ユニラジアル: 一つの環構造の中でだけ意味をなす議論（普通の数学はすべてこれ）
  • マルチラジアル: 異なる環構造（「異質な算術的正則構造」）の視点からも意味をなす議論

比喩: 異なる通貨圏の2つの国がある。ユニラジアル＝「円建て」でしか計算できない。マルチラジアル＝「為替レートが不確定でも成り立つ購買力平価のような指標」で計算する。

Q8: なぜ3種類の不定性は許容されるのか？

A: 不定性は「歪み」として不等式に吸収されるが、(1+ε) の範囲に収まるから。

不定性なぜ生じるか許容される理由
Ind1: ガロア群の内部自己同型遠アーベル復元は内部自己同型の不定性を込めてしか実行できないε に吸収
Ind2: O× の作用対数殻上で単数群の自由度が残る対数体積で制御
Ind3: 上限/下限の選択対数殻の「大きさ」の評価に幅がある明示的な定数で抑制

これらは最終的な不等式を (1+ε) の形に「緩める」が、ABC予想が言っているのはまさに (1+ε) の形なので十分。

Q9: なぜ「宇宙際 (inter-universal)」なのか？

A: 異なる「数学の宇宙」（集合論モデル）を跨ぐ操作を含むから。

  • シータリンクで繋がれた2つのホッジシアターは、ある意味で「異なる宇宙」に属する
  • これらを関係づけるために、特定の集合論モデルに依存しない「種 (Species)」の概念が必要
  • 「宇宙と宇宙の間」= Inter-universal
Q10: なぜこの理論は理解されにくいのか？

A: 複数の理由が重なっている。

  • 分量: 4本合計で約1000ページ。前提を含めると数千ページ
  • 独自用語: フロベニオイド、ホッジシアター、対数殻など、IUT以外で使われない概念が多い
  • 既存枠組みの外: スキーム論・圏論の標準的な道具立てでは記述できない操作
  • 前提知識の特殊性: 遠アーベル幾何学は世界でも少数の研究者しか扱えない
  • 検証の困難: 望月の先行研究全体の理解が必要
  • コミュニケーションの断絶: 望月と批判者の間で「何が議論の核心か」について合意がない

「2つの次元」の正体

p進局所体 k の任意の元 x ≠ 0 は一意に分解できる:

x = p^n × u     (n = v_p(x) ∈ Z,  u ∈ O_k^×)
    ~~~~   ~~~
    値群   単数群
  • 値群 k×/O_k× ≅ Z : 「pを何回含むか」= 掛け算的な次元
  • 単数群 O_k× : 「pを取り除いた残り」= 足し算的な次元

IUT理論はこの2つの次元を「解きほぐし」、掛け算側だけをシータリンクで変形する。これは古典タイヒミューラー変形 z = x+iy → ζ = Kx+iy（x方向だけK倍、y方向は固定）の算術版。

IUT I: ホッジシアターの構成

IUT理論の舞台装置を組み立てる巻。後続の議論で使うすべての数学的オブジェクトをここで構成する。

初期シータデータ (Initial Θ-data)

IUT理論全体の出発点となるデータの組:

  1. 楕円曲線 E_F: 数体 F 上の楕円曲線
  2. 素数 l ≥ 5: 理論のパラメータ
  3. 代数閉包 F̄ of F
  4. 付値の集合 V: 数体 K (= F(E[l])) の付値の部分集合
  5. 悪い還元の付値 Vbad: E が乗法的悪い還元を持つ付値

二つの対称性

F_l = Z/lZ 上の二つの対称性に分離:

対称性作用記号
加法的 (additive)F_l の足し算（±符号付き）F_l±
乗法的 (multiplicative)F_l* の掛け算F_l⊛

この分離は「環構造を解体する第一歩」。望月自身が IUT I 序文で「rough, approximate approach」と呼ぶ近似的な分離で、本当の分離はΘ-リンクで実現される。

IUT II: シータリンクの精密化

IUT I で構成したシータリンクを精密化し、テータ関数の値を正しく計算するための理論を構築する。

テータ関数の評価

楕円曲線の l-等分点 (l-torsion points) でテータ関数の値を計算する:

Θ_v = (テータ関数の l乗根の逆数)

このテータ値を、楕円曲線が悪い還元を持つ各付値 v ∈ Vbad で計算する。

Θ_gau×μ リンク

  • Θ-link (IUT I): テータ関数そのものを使う
  • Θ_gau×μ-link (IUT II): l-等分点でのテータ値を使う（より制御された変形）

マルチラジアリティの導入

シータリンクは環構造を壊すので、左側のホッジシアターの環構造は右側から見ると「異質な (alien)」構造。これを跨いで意味をなす議論＝マルチラジアル (多放射的) なアルゴリズムを設計する必要がある。

IUT III: 対数テータ格子

IUT シリーズの概念的頂点。理論の中心舞台「対数テータ格子」を構築し、マルチラジアルアルゴリズムによって最終的な見積もりの基礎を確立する。

対数テータ格子（二次元の非可換ダイアグラム）

         ↑ log               ↑ log               ↑ log
  ... ← •_{n,m+1} ← Θ ← •_{n+1,m+1} ← Θ ← •_{n+2,m+1} ← ...

  • 各 • はホッジシアター
  • 横矢印 (Θ): シータリンク（掛け算的変形、環構造を壊す）

  • 非可換: 「右→上」と「上→右」で結果が異なる！

対数殻 (Log-shell)

対数リンクの像を少し調整した集合。掛け算的構造（単数群 O_v×）を、対数を通じて足し算的構造に変換した「殻」（入れ物）。足し算と掛け算を橋渡しする入れ物として機能する。

LGP-モノイドとマルチラジアルアルゴリズム

Logarithmic Gaussian Procession Monoid (LGP-モノイド) を導入し、対数殻のテンソル積上で「テータ値の系列」を扱う。これにより、シータリンクを跨いで意味をなすマルチラジアル算法が完成する（IUT III の主定理）。

IUT IV: 最終的な不等式の導出

IUT III で得られた対数体積の見積もりを具体的に計算し、ディオファンタス幾何の定理（ABC予想を含む）を導く。

主定理: ディオファンタス不等式（定理A）

X を数体上の滑らかな射影曲線、D ⊆ X を被約因子、U_X = X \ D が双曲的曲線と仮定する。任意の正整数 d と ε > 0 に対して:

ht_{ω_X(D)} ≤ (1 + ε) · (log-diff_X + log-cond_D)

（「bounded discrepancy class」の意味で、U_X(Q̄)≤d 上の関数として成立）

ABC予想への帰着

楕円曲線 y² = x(x-a)(x-b) を適切に選ぶことで:

  • 標準高さ ↔ log(max(|a|,|b|,|c|))
  • 対数的導手 ↔ log(rad(abc))
  • 対数的判別式 ↔ 数体の判別式（有限個の例外で制御可能）

これにより定理A から ABC予想 c < K(ε) · rad(abc)1+ε が導かれる。

IUT理論全体の例え

例え1: 翻訳不可能な2つの言語

二つの国がある。A国は「足し算語」、B国は「掛け算語」を話す。普段は一つの国（環構造）で両方の言語が一体化していて翻訳不要。IUT理論は:

  1. この国を二つに分割する（ホッジシアター = 分割後の国）
  2. 二つの国の間にトンネルを掘る（シータリンク = トンネル）
  3. ただしトンネルでは「掛け算語」しか通じない（環構造の非互換性）
  4. 特殊な通訳を雇う（マルチラジアルアルゴリズム = 言語に依存しない意思疎通）
  5. 通訳の「誤差範囲」から不等式を導く（三つの不定性 → ABC予想）

例え2: 二つの地図を重ねる

同じ土地の等高線地図と地質図がある。等高線地図は高さ（足し算的構造）を、地質図は地層（掛け算的構造）を表す。普通は一枚にまとめられているが、IUT理論は二枚を別々に扱い、「微妙にずれた二枚の地図を重ねたときの歪み」から地形の制約条件（ABC予想の不等式）を導く。

個別概念の例え

環構造（足し算 + 掛け算の一体化） — 色付きの積み木

積み木には「形」（丸、四角、三角）と「色」（赤、青、緑）がある。普通の積み木は形と色が一体化していて分離できない。「赤い丸を青い丸にする」ことは、積み木を壊して作り直さないと不可能。

  • 環構造 = 形と色が不可分の積み木
  • IUT理論 = 積み木を壊して形と色を別管理にする
ホッジシアター — ジオラマ（ミニチュア模型）

数学の世界全体（スキーム論）の精巧なジオラマ。実物（環構造）を直接いじるのは危険なので、ジオラマ上で実験する。ジオラマには「足し算の棟」と「掛け算の棟」が分かれて配置されている。

シータリンク — ワームホール

二つのジオラマ（ホッジシアター）を繋ぐワームホール。ただし、このワームホールは「掛け算的な荷物」しか運べない。具体的には: ワームホールは qj² を q に変換する。これは掛け算の世界では「冪乗を変える」だけだが、足し算の世界から見ると「数そのものが変わる」ことに相当する。

マルチラジアリティ — GPS vs 方角指示
  • ユニラジアル = 「この建物を左に曲がって3ブロック先」→ 出発地点に依存
  • マルチラジアル = 「北緯35度、東経135度」→ どこから出発しても同じ場所を指す
対数テータ格子 — 碁盤の目の街

東西（シータリンク）と南北（対数リンク）に無限に広がる街。各交差点にジオラマがある。重要: 「東に行ってから北に行く」のと「北に行ってから東に行く」では、たどり着く場所は同じだが見える景色が違う（非可換性）。

対数殻 (Log-shell) — 通貨の購買力範囲

具体的な数値で「足し算と掛け算の分離」を体感する

IUT理論の核心操作——「掛け算の次元だけを動かし、足し算の次元は固定する」——を、実際の数値で見せる。抽象的な定義ではなく、手で計算できるレベルに落とす。

5進数の世界 (p=5)

5進整数とは 5 を「小さい数」と見なす世界:

通常の感覚: 125 は「大きい」、1/5 は「小さい」
5進の感覚: 125 = 5³ は「超小さい」、1/5 は「大きい」

5進付値 v₅(x) = 「x を何回 5 で割れるか」

v₅(1)   = 0    (5で割れない)
v₅(5)   = 1    (1回割れる)
v₅(25)  = 2    (2回割れる)
v₅(125) = 3    (3回割れる)
v₅(7)   = 0    (5で割れない)
v₅(50)  = 2    (50 = 2 × 5²)

「2つの次元」の正体

任意の 5 進数 x ≠ 0 は一意に分解できる:

x = 5^n × u     (n = v₅(x), u は5進単数)
    ~~~~   ~~~
    値群   単数群
    (掛け算の次元)  (足し算の次元)
数 x値群の成分 5^n単数群の成分 u分解
75⁰ = 177 = 1 × 7
255²125 = 5² × 1
505²250 = 5² × 2
355¹735 = 5¹ × 7
3755³3375 = 5³ × 3
  • 値群 = 「5を何回含むか」= 掛け算的な情報
  • 単数群 = 「5を全部取り除いた残り」= 足し算に関わる情報

なぜ「単数群 = 足し算的」なのか — 絡み合いの実例

例: 7 + 18 = 25

左辺: 7 = 5⁰ × 7,  18 = 5⁰ × 18    (どちらも値群=0、つまり単数)
右辺: 25 = 5² × 1                     (値群=2、単数=1)

2つの単数（7と18）を足しただけなのに、値群が0から2に跳ね上がった！

これが足し算と掛け算の「絡み合い」の最も単純な例:

  • 掛け算的には 7 も 18 も「5とは無関係」（値群=0）
  • でも足し算すると 7+18=25=5² で「5が2つ出てくる」
  • 足し算が掛け算の構造を変えてしまう

逆もある:

3 + 122 = 125 = 5³    (単数同士の和が値群=3を生み出す)
2 + 123 = 125 = 5³    (同上)
4 + 121 = 125 = 5³    (同上)

結論

環構造（足し算+掛け算が共存）がある限り、この絡み合いは避けられない。これがABC予想を直接証明できない根本的な理由。

IUT理論の解決法（数値で）

IUT理論はこの絡み合いを解消するため、値群側だけを変形する:

古典タイヒミューラー:    z = x + iy → ζ = K·x + iy  (x方向だけK倍)

IUT (Θ-link):           q^j² ↔ q
                         ~~~~     ~
                         値群側を伸縮  単数群側は別世界に切離

望月の言葉で「値群（掛け算的次元）はテータ関数で q → qj² に伸縮され、 単数群（足し算的次元）は固定される（正確には別世界に属するので繋がない）」。 これが古典タイヒミューラー変形の算術版。

詳細は notes/views/numerical_example_view.md を参照（テート曲線・ガウス極の数値例も含む）。

論争の核心

2018年、Fields賞受賞者 Peter Scholze と Jakob Stix が望月を訪問し、IUT III 系3.12の証明に「致命的な欠陥」があると報告。以来、数学コミュニティの多数派は「ABC予想はまだ未解決」と見なしている。

三者の立場

系3.12は正しいか不等式は非自明か核心的主張
望月 正しい はい (ABC予想を含意) マルチラジアリティと遠アーベル復元が要点。批判者は本質を理解していない
Scholze-Stix 誤り いいえ (vacuous) シータリンクを跨ぐ同一視が不当。簡略化すると新しい情報が出ない
Joshi 不完全 未決 Scholze の2021年批判は誤り、ただし望月の証明も不完全。幾何的対象を重視すべき

Scholze-Stix の批判

何が問題とされたか

  • IUT III 系3.12 の証明において、ある同一視が不当であると指摘
  • 具体的には: シータリンクを跨いで局所データを比較する際に、異なるホッジシアター間の「同一視」が正当化されていないと主張

Scholze の立場（要約）

「系3.12の証明を追うと、本質的な部分で何も新しい情報が得られない。不等式の導出において、シータリンクの非可換性から生じる困難が実際には解決されていない。」

望月の反論（要約）

「Scholze-Stixの批判は、理論の本質的な構造（特にマルチラジアリティと遠アーベル幾何学的復元）を正しく理解していないことに起因する。彼らの『簡略化』は、理論の核心的な部分を取り除いてしまっている。」

対話の断絶

両者は互いの主張を「根本的な誤解」と見なし、合意に至らなかった。数学における前例のない事態 — 査読付き雑誌に掲載されたにも関わらず、フィールズ賞受賞者（Scholze）が証明の正当性を認めないという状況。

Kirti Joshi のアプローチ

第三の道

Joshi（アリゾナ大学）は2024年以降、独自の視点からIUT理論を再構成しようとしている。

  1. Scholze の2021年の批判は数学的に誤りと主張
  2. しかし望月の証明も不完全と認める
  3. 望月の「数体のタイヒミューラー理論が存在する」という直観は正しいが、その存在証明には望月の遠アーベル幾何学的手法だけでは不十分
  4. 幾何学的・算術的対象の「優位性」(primacy) を重視すべき

シータリンク (Θ-link)
二つのホッジシアターを繋ぐ操作。q^j² → q の対応。環構造を壊す
対数リンク (log-link)
p進対数写像を通じてホッジシアターを縦に繋ぐ
対数テータ格子
シータリンク（横）と対数リンク（縦）で繋いだ二次元格子
対数殻 (log-shell)
p進対数の像を調整した集合。足し算と掛け算を橋渡し
マルチラジアル
異なる環構造の視点からも意味をなすアルゴリズム
ユニラジアル
一つの環構造の中でだけ意味をなすアルゴリズム
算術的正則構造
環構造のこと。複素解析の正則構造の算術版
フロベニオイド
数体の掛け算的部分を抽象化した圏論的構造
共役同期
異なるl-等分点の局所ガロア群が整合的に同一視されること
不定性 (indeterminacy)
マルチラジアル算法適用時に避けられない「ずれ」
LGP-モノイド
Logarithmic Gaussian Procession Monoid。対数殻版のガウスモノイド
種 (species)
「数学的対象の型」を集合論的公式で形式化したもの
変異 (mutation)
種の間の対応。関手の集合論的類似物
核 (core)
変異の図式から不変な部分を取り出す操作
