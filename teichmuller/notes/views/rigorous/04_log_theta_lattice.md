# 対数テータ格子 [IUT III, §1]

---

## 定義 [IUT III, Definition 1.4, Definition 3.8]

### 格子の構造

Z × Z で添字づけられた Θ±ell NF-ホッジシアターのダイアグラム:

```
{n,m HT^{Θ±ell NF}}_{(n,m)∈Z²}
```

### 矢印の定義

**水平矢印** (n,m) → (n+1,m):
- Θ^{×μ}_{LGP}-リンク（シータリンクの対数殻版）

**垂直矢印** (n,m) → (n,m+1):
- log-リンク（各 v ∈ V^non で p_v 進対数写像を適用）

### 非可換性

**重要**: このダイアグラムは非可換。すなわち:

```
(log-link) ∘ (Θ-link) ≠ (Θ-link) ∘ (log-link)
```

この非可換性が理論の本質に関わる。

---

## Log-リンクの厳密な定義 [IUT III, Proposition 1.3]

### 各非アルキメデス付値 v ∈ V^non での構成

```
log_v: O_{K_v}^× → K_v
```

p_v 進対数写像。ここで:
- O_{K_v}: K_v の整数環
- O_{K_v}^×: その単数群

### 対数殻 (Log-shell) [AbsTopIII]

```
I_v := (p_v^{-1} · log_v(O_{K_v}^×)) ∩ O_{K_v}  ⊆ K_v
```

対数殻 I_v は:
- 足し算的な対象（K_v の部分集合）
- log-リンクの反復に対して不変（核心的性質）
- 対数体積が明示的に計算可能

### Log-リンクが誘導する同型

Log-リンクは基底にある D-Θ±ell NF-ホッジシアター間の同型を誘導するが、
フロベニオイド的データは保存しない:

```
n,m HT --[log]--> n,m+1 HT
  D-data: 同型で写る
  Frobenioid-data: 写らない（掛け算的構造が変わる）
```

## 格子の各部分の役割

### 一つの垂直列（固定した n の列）

```
... → n,m-1 HT → n,m HT → n,m+1 HT → ...
         log         log
```

log-リンクの反復。対数殻 I_v はこの反復に対して不変（= core）。

### 一つの水平列（固定した m の列）

```
... → n-1,m HT → n,m HT → n+1,m HT → ...
         Θ           Θ
```

Θ-リンクの反復。テータ値の q^{j²} ↔ q の対応の連鎖。

### 格子全体

縦（log）と横（Θ）の非可換な組み合わせにより:
- テータ値の情報が対数殻を通じて「足し算的に」制約される
- この制約が最終的な不等式の源泉となる

## フロベニウス写像とのアナロジー [IUT IV, §3]

### 正標数のフロベニウス

標数 p のスキーム S に対する フロベニウス写像 S → S^{(p)}:
- 位相空間は同じだが、正則関数を p 乗する
- 反復: S → S^{(p)} → S^{(p²)} → ...
- 核 (core): 完全化 S^{pf} およびエタールサイト S^{ét}

### IUT理論版

| フロベニウス | IUT理論 |
|------------|---------|
| S → S^{(p)} | Θ-リンク（水平矢印）|
| 正則関数の p 乗 | q → q^{j²} の対応 |
| 完全化 S^{pf} | log-リンクの反復による対数殻 |
| エタールサイト S^{ét} | D-データ（遠アーベル的な不変量）|

この類似が「対数テータ格子 = 正標数フロベニウスの数論版」という直感の根拠。
