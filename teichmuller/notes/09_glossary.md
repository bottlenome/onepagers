# 用語集

IUT理論に登場する主要用語を、一般向けの説明付きで整理する。

---

## 基礎的な数学用語

| 用語 | 読み | 説明 |
|------|------|------|
| **素数** | そすう | 1とそれ自身でしか割り切れない2以上の整数。例: 2, 3, 5, 7, 11... |
| **素因数分解** | そいんすうぶんかい | 整数を素数の積として表すこと。例: 12 = 2² × 3 |
| **体 (field)** | たい | 四則演算（加減乗除）が自由にできる数の集合。Q（有理数）、R（実数）、C（複素数）など |
| **環 (ring)** | かん | 足し算と掛け算ができる（割り算は不要な）数の集合。整数 Z が典型例 |
| **群 (group)** | ぐん | 一つの演算（とその逆演算）ができる集合。対称性を記述する |
| **圏 (category)** | けん | 対象（もの）と射（矢印）の集まり。数学的構造間の関係を記述する枠組み |
| **同型 (isomorphism)** | どうけい | 構造を完全に保つ対応。数学的に「同じもの」とみなせる |
| **関手 (functor)** | かんしゅ | 圏から圏への「構造を保つ対応」|

## 数論・代数幾何の用語

| 用語 | 読み | 説明 |
|------|------|------|
| **数体 (number field)** | すうたい | 有理数 Q に代数的な数を付け加えた体。例: Q(√2) |
| **楕円曲線 (elliptic curve)** | だえんきょくせん | y² = x³ + ax + b の形の曲線。点同士の「足し算」が定義できる |
| **l-等分点 (l-torsion point)** | — | 楕円曲線上で l 回「足す」と原点に戻る点 |
| **ガロア群 (Galois group)** | — | 体の拡大の対称性を記述する群 |
| **基本群 (fundamental group)** | きほんぐん | 空間のループ構造を記述する群。穴の数を代数的に捉える |
| **エタール基本群** | — | 基本群の代数幾何版。数論的な情報を含む |
| **付値 (valuation)** | ふち | 数の「大きさ」を測る方法。各素数 p ごとに一つ、実数用にも存在 |
| **p進数 (p-adic number)** | — | 素数 p に基づく距離で完備化した数体系 |
| **スキーム (scheme)** | — | 代数幾何の基本的な対象。環から構成される幾何学的空間 |
| **良い還元 / 悪い還元** | — | 楕円曲線を mod p で見たときの振る舞い |
| **q-パラメータ** | — | テート曲線（p進的な楕円曲線）のパラメータ |

## IUT理論固有の用語

| 用語 | 読み | 説明 |
|------|------|------|
| **初期シータデータ (Initial Θ-data)** | — | IUT理論の出発点となるデータの組（楕円曲線、素数 l、付値の集合など）|
| **ホッジシアター (Hodge theater)** | — | 通常のスキーム論の「ミニチュアモデル」。足し算と掛け算の対称性を分離して保持 |
| **シータリンク (Θ-link)** | — | 二つのホッジシアターを繋ぐ操作。q^{j²} → q の対応。環構造を壊す |
| **対数リンク (log-link)** | — | p進対数写像を通じてホッジシアターを縦に繋ぐ操作 |
| **対数テータ格子 (log-theta-lattice)** | — | ホッジシアターをシータリンク（横）と対数リンク（縦）で繋いだ二次元格子 |
| **対数殻 (log-shell)** | — | p進対数の像を調整した集合。足し算と掛け算を橋渡しする「入れ物」|
| **マルチラジアル (multiradial)** | — | 異なる環構造の視点からも意味をなすアルゴリズムや概念 |
| **ユニラジアル (uniradial)** | — | 一つの環構造の中でだけ意味をなすアルゴリズムや概念 |
| **算術的正則構造** | — | 環構造（足し算+掛け算）のこと。複素解析の正則構造の算術版 |
| **フロベニオイド (Frobenioid)** | — | 望月が導入した圏論的構造。数体の掛け算的部分を抽象化 |
| **共役同期 (conjugate synchronization)** | — | 異なるl-等分点の局所ガロア群が整合的に同一視されること |
| **不定性 (indeterminacy)** | — | マルチラジアルアルゴリズム適用時に生じる避けられない「ずれ」|
| **LGP-モノイド** | — | Logarithmic Gaussian Procession Monoid の略。対数殻版のガウスモノイド |
| **種 (species)** | — | 「数学的対象の型」を集合論的公式で形式化したもの |
| **変異 (mutation)** | — | 種の間の対応。関手の集合論的類似物 |
| **核 (core)** | — | 変異の図式から不変な部分を取り出す操作 |

## アナロジー的用語（古典との対応）

| 古典的概念 | IUT での対応物 | 共通する本質 |
|-----------|--------------|-------------|
| リーマン面 | 数体＋楕円曲線 | 変形する対象 |
| 正則構造 | 環構造 | 保ちたい/変形する構造 |
| 擬等角変形 | シータリンク | 構造を歪める操作 |
| タイヒミューラー空間 | 対数テータ格子 | 変形のパラメータ空間 |
| ガウス分布 e^{-x²} | テータ値 q^{j²} | 二次形式的な構造 |
| √π | ω_E（コタンジェント束）| 変形の「結果」の値 |
