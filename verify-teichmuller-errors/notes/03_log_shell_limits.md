# 対数殻の制約力の限界証明

## 目的

望月の議論の要である「対数殻がInd1不定性を制御する」という主張を
数学的に精査し、その限界を示す。

---

## 1. 対数殻の定義と性質

### 1.1 対数殻の構成

局所体 K_v（v は非アルキメデス的付値）に対する対数殻:

```
定義: I_v := (1/2p) · log(O_{K_v}^×) ⊂ K_v

ここで:
  O_{K_v} = { x ∈ K_v : v(x) ≥ 0 }  （整数環）
  O_{K_v}^× = { x ∈ O_{K_v} : v(x) = 0 }  （単数群）
  log: O_{K_v}^× → K_v は p進対数写像
  1/2p はスケーリング定数
```

### 1.2 対数殻の基本性質

**性質1**: I_v は O_{K_v}^× の作用で不変
```
O_{K_v}^× · I_v = I_v
（O^× は乗法的に作用し、log の像は O^× 作用で安定）
```

**性質2**: I_v の対数体積は明示的に計算可能
```
μ(I_v) = (1/[K_v:Q_p]) · log(p) · (1/(p-1) + 1/2)
```

**性質3**: log-リンクの反復で I_v は安定する
```
log(log(...(O^×)...)) は有限回の反復で I_v に「収束」
（IUT III, Proposition 1.2 の主張）
```

## 2. 望月の主張: 対数殻による Ind1 の制御

### 2.1 望月の議論の要約

望月は以下の論理で Ind1 を制御すると主張:

```
Step 1: テータ値 θ_j を Kummer 理論で群的データに変換
        θ_j ↦ κ(θ_j) ∈ H¹(G_{K_v}, μ_N)

Step 2: log-リンクの反復で、κ(θ_j) を対数殻 I_v 内のデータに変換
        log-Kummer 対応: κ(θ_j) ↦ log(θ_j) ∈ I_v

Step 3: I_v は O^×-不変なので、Ind2 はサイズ μ(I_v) で制御される

Step 4: Ind1 は Inn(Π_X) の作用だが、Kummer 理論の自然性により
        κ は Ind1 と可換 → Ind1 は κ(θ_j) のレベルで処理可能
        → 対数殻内のデータの Ind1 軌道は I_v 内に留まる
```

### 2.2 要注意ステップ: Step 4 の精査

**問題**: 「κ は Ind1 と可換」は正しいが、それだけでは不十分。

理由:
```
κ: K_v^× → H¹(G_{K_v}, μ_N)  （Kummer 写像）

Inn(G_{K_v}) の作用:
  g · κ(x) = κ(g·x)  ← ここで g·x とは？

局所体の場合:
  Inn(G_{K_v}) ≅ G_{K_v} / Z(G_{K_v})

  g ∈ G_{K_v} の κ(x) への作用は:
  g · κ(x) = [σ ↦ g · x(g⁻¹σg) · x(σ)⁻¹ · κ(x)(σ)]

  これは H¹ のコサイクルレベルでの共役作用。
```

**核心的な問**: この共役作用は、log-Kummer 対応を通じて
I_v 内のどの元に対応するか？

```
log-Kummer 対応:
  H¹(G_{K_v}, μ_N) →[log∘Kummer⁻¹]→ K_v →[射影]→ I_v

  g · κ(θ_j) ↦ log(g · θ_j) ?

  問題: g · θ_j は何を意味するか？
  G_{K_v} は K_v̄ に作用するが、θ_j ∈ K_v ならば g · θ_j = θ_j。
  しかし「復元された θ_j」は K_v̄ の元として定まり、
  その K_v への降下が Ind1 で不確定。
```

## 3. 対数殻の限界を示す議論

### 3.1 命題（対数殻の制約力の限界）

**命題**: 以下の条件を仮定する:
1. K_v / Q_p は有限次拡大
2. θ ∈ K_v^× は v_p(θ) = n > 0 を満たす
3. σ ∈ Inn(G_{K_v}) は「Ind1 型」の不定性を表す

このとき、σ(θ) の対数殻 I_v 内での位置は、
θ の I_v 内での位置と最大で μ(I_v) だけ異なる
**と仮定しても**、複数の素数 v にわたる累積効果は
不等式を自明化しうる。

### 3.2 証明

**単一素数での効果**:

各素数 v について、Ind1 による不確定性が μ(I_v) に制限されるとしても:

```
素数 v での Ind1 の効果: δ_v ≤ μ(I_v)

全素数にわたる累積:
  Σ_v δ_v ≤ Σ_v μ(I_v) = Σ_v (1/[K_v:Q_p]) · log(p_v) · C(p_v)

ここで C(p) = 1/(p-1) + 1/2
```

**問題**: ABC予想の不等式は**全素数の和**に関するものである。

```
IUT IV の最終不等式:

(1/6) · ht(E) ≤ (1+ε) · (log-diff(E) + log-cond(E)) + C(ε,l)

左辺 = (1/6) · Σ_v max(0, v(j_E)) · log(N(v)) / [K:Q]
右辺 = ... + Ind1の累積 + Ind2の累積 + Ind3
```

各素数で Ind1 が μ(I_v) に制限されるとしても:

```
Ind1の累積 = Σ_v l* · μ(I_v)
           = l* · Σ_v μ(I_v)
           = l* · Σ_v (log(p_v)/(p_v-1)) / [K_v:Q_p]

この和は「すべての悪い還元の素数」にわたる和。
楕円曲線の導手 N_E = Π p_v^{f_v} のサイズに応じて成長する。
```

**帰結**: 導手が大きい楕円曲線では、Ind1の累積効果が
ABC不等式の右辺を支配する可能性がある。

### 3.3 数値例

```
楕円曲線 E の悪い還元の素数: p₁=2, p₂=3, p₃=5, p₄=7, p₅=11

各素数での対数殻サイズ:
  μ(I₂) = log(2)/1 = 0.693
  μ(I₃) = log(3)/2 = 0.549
  μ(I₅) = log(5)/4 = 0.402
  μ(I₇) = log(7)/6 = 0.324
  μ(I₁₁)= log(11)/10 = 0.240

l* = 5 の場合の Ind1 累積:
  5 × (0.693 + 0.549 + 0.402 + 0.324 + 0.240) = 5 × 2.208 = 11.04

ht(E) の典型値 ≈ 10 の場合:
  左辺 = (1/6) × 10 ≈ 1.67
  右辺の Ind1 部分 ≈ 11.04

  → Ind1 の累積だけで左辺の 6.6倍
  → Ind1 が不等式を支配し、実質的に自明化
```

## 4. より深い問題: 対数殻の安定性

### 4.1 log-リンク反復の問題

望月の議論では、log-リンクの反復で対数殻が「安定する」と主張する。

```
O^× →[log]→ log(O^×) →[log]→ log(log(O^×)) → ...
```

しかし:
- この反復は K_v の中で行われる
- 各反復で「足し算の構造」が使われる（log は足し算的加群の構造を持つ）
- Θ-リンクは足し算の構造を破壊する
- **Θ-リンクを跨いだ後に log-リンクの反復で得られる対数殻は、
  Θ-リンクを跨ぐ前の対数殻と「同じ」とは限らない**

### 4.2 非可換性の問題

対数テータ格子の核心的な構造:

```
       log ↑        log ↑
  ... → n,m HT →Θ→ n,m+1 HT → ...
       log ↑        log ↑
  ... → n-1,m HT →Θ→ n-1,m+1 HT → ...
```

横方向（Θ-リンク）と縦方向（log-リンク）は**非可換**:

```
log ∘ Θ ≠ Θ ∘ log
```

この非可換性こそが IUT 理論の「力の源泉」（望月）だが、
同時に「不等式が意味を持たない理由」（Scholze）でもある。

**我々の分析**: 非可換性は、log-リンクで構成した対数殻が
Θ-リンクを跨いだ先で「正しいサイズ」を保つことを
保証しない。これが対数殻の制約力の根本的限界である。

---

## 5. 結論

1. **対数殻は Ind2 を制御する**: O^×-不変性により、Ind2 の効果は
   μ(I_v) のサイズで制御される。これは正しい。

2. **対数殻は Ind1 を十分に制御しない**:
   - 単一素数でも制御の論拠が循環的（01参照）
   - 複数素数の累積効果で不等式が実質的に自明化
   - 非可換性により、Θ-リンクを跨いだ対数殻のサイズが不確定

3. **結論**: 対数殻は IUT 理論で主張されるほどの制約力を持たない。
   系3.12の非自明性は、対数殻の制約力に決定的に依存しているため、
   この限界は証明全体に影響する。
