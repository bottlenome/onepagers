# 三つの不定性の定量的分析

## 目的

IUT理論の三つの不定性 (Ind1, Ind2, Ind3) が、系3.12の不等式に
与える定量的影響を分析する。
特に「不定性が不等式を自明化するか否か」を数値的に検証する。

---

## 1. 三つの不定性の定量的モデル

### 1.1 セットアップ

以下のパラメータで作業する:
- p: 素数（テート曲線の定義体の素数）
- q = p^a: テート曲線のqパラメータ (a = v_p(q) > 0)
- l: 奇素数（l-等分点のレベル）
- l* = (l-1)/2

テータ値列: {q^{j²} : j = 1, ..., l*}
付値列: {a·j² : j = 1, ..., l*}

### 1.2 Ind1 の定量的モデル

**Ind1 = 内部自己同型の不定性**

遠アーベル復元で群 Π_X から環を復元するとき、
Inn(Π_X) ≅ Π_X / Z(Π_X) が作用する。

局所体 K_v の絶対ガロア群 G_{K_v} の場合:
- |Inn(G_{K_v})| は非可算（プロ有限群の共役類は無限）
- ただし付値 v_p は Inn の作用で不変

**Ind1が動かすもの**: 環の元の「名前」（どの元がどの元に対応するか）
**Ind1が動かさないもの**: 付値（v_p の値）

定量的効果:
```
Ind1の作用後のテータ値 θ_j:
  v_p(θ_j) = a·j²  （付値は不変）
  θ_j の「単数部分」= 任意の単数 u ∈ O_{K_v}^×  （完全に不確定）

つまり: θ_j = p^{a·j²} × u_j,  u_j ∈ O_{K_v}^× は不明
```

### 1.3 Ind2 の定量的モデル

**Ind2 = O^× 作用の不定性**

対数殻 I_v 上で O_{K_v}^× が掛け算で作用する。

対数殻の定義:
```
I_v = { x ∈ K_v : v_p(x) ≥ -1/(p-1) } / O_{K_v}^×
    （ただし、これは対数写像 log: O_{K_v}^× → K_v の像）
```

対数殻の「サイズ」（対数体積）:
```
μ(I_v) = log(p) / (p-1)    （正規化された対数体積）
```

p=5 の場合:
```
μ(I_5) = log(5) / 4 ≈ 0.402
```

**Ind2が動かすもの**: 対数殻内の元の位置
**Ind2が動かさないもの**: 対数殻に「入っているか否か」

定量的効果:
```
Ind2により、テータ値の対数殻内での位置が O^× 軌道に沿って動く。
対数体積への影響: ±μ(I_v) の範囲で変動
```

### 1.4 Ind3 の定量的モデル

**Ind3 = 上下限の不定性**

対数体積の評価における上界と下界の差。

```
Ind3 の効果 ≤ C₃ (定数、l と素数の集合に依存)
```

IUT IV の計算では:
```
C₃ ≈ (1 + 12·ε) × (log-diff + log-cond) + 定数
```

## 2. 不定性の累積効果

### 2.1 望月の主張する累積モデル

望月は三つの不定性を**独立に**処理できると主張:

```
対数体積の見積もり:

|log(Θ)| = Σ_{j=1}^{l*} v_p(q^{j²}) × log(p) / [K_v : Q_p]
          = a × (Σ j²) × log(p) / [K_v : Q_p]

不定性による補正:
  Ind1: 付値を変えないので 0
  Ind2: ±l* × μ(I_v)    (各 j に対して μ(I_v) の幅)
  Ind3: ±C₃

最終不等式:
  |log(q)| ≤ |log(Θ)| + l* × μ(I_v) + C₃

  a·log(p) ≤ a·(Σj²)·log(p)/[K:Q_p] + l*·log(p)/(p-1) + C₃
```

Σj² ≈ l³/24 なので、l を十分大きく取れば左辺 << 右辺で非自明。

### 2.2 Scholze が主張する累積モデル

Scholzeは Ind1 の効果が望月の評価より遥かに大きいと主張:

```
Ind1 の真の効果（Scholzeモデル）:

遠アーベル復元で群 → 環 を復元する際、
内部自己同型は「付値は保存するが、どの元がテータ値に対応するかを失う」。

具体的には:
  世界Aのテータ値 θ_j = q^{j²} は、世界Bから見ると
  「v_p = a·j² を持つ任意の元」でしかない。

  これを対数殻で「閉じ込める」には、
  θ_j の対数殻内での位置が Ind1 で動かないことが必要。

  しかし Ind1（群レベルの操作）と対数殻（環レベルの構造）の間には
  次の問題がある:

  Inn(G_{K_v}) の作用は O_{K_v}^× を O_{K_v}^× に写すが、
  対数写像 log: O_{K_v}^× → I_v との可換性が保証されない。
```

**Scholzeモデルの帰結**:

```
Ind1 の効果が対数殻に閉じ込められないなら:

  Ind1 の見積もり ≈ ∞  （任意の値を取りうる）

  最終不等式: |log(q)| ≤ |log(Θ)| + ∞ = ∞

  → 自明（何も言っていないのと同じ）
```

## 3. 数値的検証

### 3.1 パラメータ設定

```
p = 5, a = v₅(q) = 2, l = 11, l* = 5
```

### 3.2 望月モデルでの計算

```
テータ値の対数体積:
  |log(Θ)| = 2 × (1+4+9+16+25) × log(5) = 2 × 55 × 1.609 = 176.99

q-パラメータの対数体積:
  |log(q)| = 2 × log(5) = 3.219

Ind2 の補正:
  l* × μ(I₅) = 5 × 0.402 = 2.012

望月モデルの不等式:
  3.219 ≤ 176.99 + 2.012 + C₃

  → 明らかに成立（非自明）
  → 歪み率 = 176.99 / 3.219 ≈ 55
```

### 3.3 Scholzeモデルでの計算

```
テータ値の対数体積: 176.99 （同じ）
q-パラメータの対数体積: 3.219 （同じ）

Ind1 の補正（Scholzeモデル）:
  Ind1 は†HT側のテータ値と‡HT側のq-パラメータの対応関係を破壊。
  対応関係が失われると、テータ値列 {q^{j²}} の情報は
  「v₅ = 2,8,18,32,50 を持つ何かの元の集合」にしかならない。

  この情報から導ける不等式:
  「v₅(q) ≤ max{2,8,18,32,50} = 50」
  → これは v₅(q) = 2 に対して成立するが、
     ABC予想の形の不等式（高さの上界）にはならない。

  なぜなら: 上の不等式は q の値に依存しない。
  q = 5^100 でも max{100, 400, 900, 1600, 2500} = 2500 で
  「v₅(q) ≤ 2500」が成立してしまう。
  → 自明な不等式
```

### 3.4 自明化の閾値

**問い**: Ind1がどの程度制御されれば不等式が非自明になるか？

```
必要条件: Ind1による補正 < |log(Θ)| - |log(q)|
         = 176.99 - 3.219 = 173.77

対数殻のサイズ: l* × μ(I₅) = 2.012

比率: 173.77 / 2.012 ≈ 86.3

つまり: 対数殻は Ind1 の効果の 1/86 しかカバーしない。
       残りの 85/86 は「Ind1 が対数殻に閉じ込められる」
       という望月の主張に依拠する。
```

この「85/86」のギャップが、Scholze-Stix 批判の定量的な核心である。

---

## 4. 結論

### 4.1 二つのモデルの比較

| 項目 | 望月モデル | Scholzeモデル |
|------|-----------|-------------|
| Ind1の効果 | 0（付値不変なので） | ∞（対応関係が壊れる） |
| Ind2の効果 | l* × μ(I_v) | l* × μ(I_v) |
| Ind3の効果 | C₃ | C₃ |
| 不等式 | 非自明 | 自明 |
| 前提 | 対数殻がInd1を制御 | 対数殻はInd1を制御しない |

### 4.2 判定

**望月モデルの問題点**: Ind1の効果を「0」と評価する根拠が不十分。
「付値が不変」であることは「不等式が非自明」であることの十分条件ではない。

**Scholzeモデルの妥当性**: Ind1が対応関係を破壊するという分析は、
遠アーベル幾何学の基本的性質（復元の不定性 = Inn(Π)）と整合的。

**結論**: 不定性の定量分析は、Scholze-Stix の批判を支持する。
