# Θ-リンクと Θ_gau^{×μ}-リンク [IUT I–II]

---

## Θ-リンクの定義 [IUT I, Definition 3.8]

### 構成

二つの Θ±ell NF-ホッジシアター †HT, ‡HT に対して、
Θ-リンクは以下の同一視として定義される:

各 v ∈ V^bad に対し:

```
†Θ_v  ↦  ‡q_v
```

ここで:
- †Θ_v: †HT 側のテータ関数（の適切な冪根の逆数）
- ‡q_v: ‡HT 側のテート曲線の q-パラメータ

### 環構造の非互換性

この同一視は**フロベニオイド的な同型**（掛け算的構造の同一視）であり、
環としての同型（足し算を含む同型）ではない。

すなわち: †HT の足し算と ‡HT の足し算は、Θ-リンクを通じて対応しない。

## Θ_gau^{×μ}-リンク [IUT II, Definition 4.9]

### テータ値を用いた精密化

l-等分点 j = 1, ..., l* でのテータ値:

```
Θ_v^j := Θ_v(√-1 · q_v^j)
```

テート曲線の明示的なテータ関数の級数展開 [EtTh, Prop 1.4] により:

```
Θ_v(u) = √-1 · Σ_{m∈Z} (-1)^m · q_v^{(m+1/2)²/2} · u^{m+1/2}
```

ここから、v ∈ V^bad における評価値は:

```
Θ_v^j ∼ q_v^{j²}  (主要項として)
```

### Θ_gau^{×μ}-リンクの定義

```
†{q_v^{j²}}_{j=1,...,l*}  ←→  ‡{q_v^{j²}}_{j=1,...,l*}
```

ただし ×μ は「単数の μ-部分（有限位数の元）を法として」の意。

---

## 核心的な対応 (∗KEY)

パノラミック概要で望月が強調する「キー対応」:

```
{q_v^{j²}}_{j=1,...,l*}  →  q_v
```

すなわち: テータ値の列 q^{1}, q^{4}, q^{9}, ..., q^{l*²} を
単一の q-パラメータに「潰す」操作。

### この対応の意味

左辺（テータ値の列）は IUT III の定理B で対数体積が見積もられる。
右辺（q-パラメータ）は楕円曲線の高さ（= ABC予想の左辺）に対応する。

∗KEY 対応が「数体の自己同型のように」機能するならば、
左辺の見積もりから右辺の見積もり（= ABC予想の不等式）が導かれる。

---

## 関数体とのアナロジー

∗KEY 対応の動機は数体と関数体のアナロジー:

| 数体 | 関数体 F_p(t) |
|------|--------------|
| q-パラメータ q_v | 不定元 t |
| q^{j²} → q の対応 | t → t^a の対応（体の自己準同型）|

関数体では t → t^a は正当な体の自己同型を与える。
数体では対応する「自己同型」は存在しない — だからこそ環構造を壊す必要がある。
