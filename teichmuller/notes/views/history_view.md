# 歴史ビュー — IUT理論に至る数学の流れ

このビューでは「この理論はどこから来たのか」を時間軸で追う。

---

## 前史: ABC予想の誕生まで

| 年 | できごと |
|----|---------|
| **1637** | フェルマーが「最終定理」を余白に記す |
| **1859** | リーマンがゼータ関数の論文を発表 |
| **1882** | デデキント・ウェーバーが「数体と関数体のアナロジー」を確立 |
| **1913** | ラマヌジャンがハーディに手紙を書く。テータ関数の深い性質 |
| **1939** | タイヒミューラーがリーマン面の変形理論を創始 |
| **1960年代** | グロタンディークがスキーム論を確立。現代代数幾何の基盤 |
| **1970年代** | ファルティングスの先駆的研究。アラケロフ幾何の発展 |
| **1983** | ファルティングスがモーデル予想を証明 |
| **1985** | Oesterlé と Masser が **ABC予想** を独立に提唱 |
| **1986** | Szpiro が楕円曲線版の予想（Szpiro予想）を定式化 |
| **1995** | ワイルズが**フェルマーの最終定理**を証明（谷山・志村予想経由）|

## IUT理論公開後

| 年 | できごと |
|----|---------|
| **2012年8月** | 論文公開。世界の数学者に衝撃が走る |
| **2012–2014** | 数名の数学者（星裕一郎、山下剛など）が理論の理解を試みるセミナーを開催 |
| **2015** | Oxford で IUT 理論に関する国際ワークショップ開催 |
| **2015–2017** | 理解の進展は遅い。「証明を理解した」と公言する研究者はごく少数 |
| **2017** | Scholze がブログで系3.12への疑問を表明 |
| **2018年3月** | **Scholze と Stix が京都を訪問**。望月と1週間にわたり議論 |
| **2018年5月** | Scholze-Stix が「系3.12に致命的な欠陥がある」と報告書を公開 |
| **2018年** | 望月が反論文書を公開。対話は平行線に |
| **2019** | Fesenko らが IUT 理論の正当性を支持する文書を公表 |
| **2020** | 論文の改訂版が公開される |
| **2021年3月** | **PRIMS に4本の論文が掲載**（特別号として出版）|
| **2021年** | Scholze が再度、系3.12の議論が不正確であると主張。数学コミュニティで議論再燃 |
| **2022年4月** | NHKスペシャル「数学者は宇宙をつなげるか？」放送 |
| **2023** | RIMS で IUT Summit 開催 |
| **2024** | **Kirti Joshi** が独自アプローチの論文を発表。Scholze との対話を経て新見解を提示 |
| **2024年3月** | 望月が "Essential Logical Structure of IUT" を公開 |
| **2025** | Joshi が "Final Report on the Mochizuki-Scholze-Stix Controversy" を公開。決着には至らず |

## 望月の研究スタイルの特徴

- プレプリントサーバー (arXiv) を使わず、自身のウェブサイトのみで公開
- 国際学会にほとんど出席しない
- 論文の解説のために他国を訪問しない
- 自身のウェブサイトで詳細な反論や解説を公開するスタイル
- この孤立的なスタイルが理論の受容の遅さの一因とも指摘される
