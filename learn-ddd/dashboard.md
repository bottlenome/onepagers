# DDD実践入門

ドメイン駆動設計（DDD）の核心をブラウザ上でインタラクティブに学べるワンページャー。

## 概要

- 種別: doc
- 状態: done
- ファイル: `index.html`

## 機能

- DDDの全体像（戦略的設計 vs 戦術的設計のマップ）
- ユビキタス言語の解説と悪い例→良い例の対比
- 境界づけられたコンテキスト（同じ概念が文脈で変わるインタラクティブ例示）
- エンティティ vs 値オブジェクト比較と判定フローチャート
- 集約パターン（Order集約の構造図とコード例）
- ドメインイベント（イベントフローアニメーション）
- リポジトリパターン（インターフェースと実装の分離）
- レイヤードアーキテクチャ（4層の展開式図解）
- ドメインモデリングクイズ（8問の分類演習）

## タスク

- [x] 全体構成の設計
- [x] 戦略的設計セクション（ユビキタス言語、境界づけられたコンテキスト）
- [x] 戦術的設計セクション（エンティティ、値オブジェクト、集約、イベント、リポジトリ）
- [x] レイヤードアーキテクチャ解説
- [x] インタラクティブ演習（分類クイズ）
- [x] HTML作成（1ファイル完結）
