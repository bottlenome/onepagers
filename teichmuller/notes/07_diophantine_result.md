## 対数体積の計算 (§1)

### IUT IIIからの引き継ぎ

IUT III 定理Bで得られたLGP-モノイドの対数体積の抽象的な見積もりを、
具体的な数論的不変量で表現する。

### 計算に使う道具

| 不変量 | 意味 |
|--------|------|
| **判別式 (different)** | 数体の拡大の「分岐」を測る量 |
| **導手 (conductor)** | 楕円曲線の「悪い還元の程度」を測る量 |
| **対数的判別式 (log-diff)** | 判別式の対数版 |
| **対数的導手 (log-cond)** | 導手の対数版 |
| **標準高さ (canonical height)** | 楕円曲線の「大きさ」を測る量 |

### 計算の流れ

1. 各非アルキメデス付値での対数殻のテンソル積の対数体積を計算（命題1.1–1.4）
2. アルキメデス付値での対数体積を計算（命題1.5、技術的にはずっと簡単）
3. 素数の分布に関する古典的な見積もり（命題1.6）を適用
4. 重み付き平均の評価（命題1.7）
5. 楕円曲線の基本的性質のまとめ（命題1.8）
6. これらを組み合わせて**明示的なディオファンタス不等式**を得る（定理1.10）

## 主定理: ディオファンタス不等式

### 定理A（論文の主結果）

X を数体上の滑らかな射影曲線、D ⊆ X を被約因子、U_X = X \ D とし、
ω_X を X の標準層とする。U_X が双曲的曲線であると仮定する。

任意の正整数 d と正実数 ε > 0 に対して:

```
ht_{ω_X(D)} ≤ (1 + ε)(log-diff_X + log-cond_D)
```

（「bounded discrepancy class」の意味で、U_X(Q̄)^{≤d} 上の関数として成立）

### 各項の意味

- **ht_{ω_X(D)}**: 標準高さ — 曲線上の点の「大きさ」
- **log-diff_X**: 対数的判別式 — 定義体の「複雑さ」
- **log-cond_D**: 対数的導手 — 悪い還元の「程度」
- **(1 + ε)**: 任意に1に近い定数

### 直感的な意味

「双曲的曲線上の有理点の高さは、判別式と導手で上から抑えられる」

## ABC予想への帰着

### 定理AからABC予想へ

楕円曲線 y² = x(x-a)(x-b) を適切に選ぶことで:

- 標準高さ ↔ log(max(|a|,|b|,|c|)) に対応
- 対数的導手 ↔ log(rad(abc)) に対応
- 対数的判別式 ↔ 数体の判別式（有限個の例外で制御可能）

したがって定理Aは以下を含意する:

```
log c ≤ (1 + ε) · log rad(abc) + 定数
```

これはまさに **ABC予想** の形。

### 他の帰結

| 予想 | 状態 |
|------|------|
| **ABC予想** | 定理Aの特殊ケース |
| **Szpiro予想** | 楕円曲線に対する定理Aの帰結 |
| **Vojta予想（双曲曲線）** | 定理Aそのもの |
| **フェルマーの最終定理（有効版）** | ABC予想からの帰結 |

## 集合論的基礎 (§3)

### 「種」(Species) の理論

IUT IV の §3 は数学の基礎論に関する議論で、ディオファンタスの結果とは独立。

#### なぜ基礎論が必要か

IUT理論では「異なるモデルの通常のスキーム論を貼り合わせる」操作を行う。
これは通常の圏論の枠組みには収まらない。

#### Species（種）の概念

「数学的対象の型」を集合論的公式（formula）で形式化したもの。

- 圏は特定のモデルに束縛されるが、Speciesは任意の集合論モデルで意味をなす
- Mutation（変異）: Species間の「関手の類似物」
- Core（核）: 変異の図式から不変な部分を取り出す操作

#### 「宇宙際」の由来

まさにこの「複数の集合論的宇宙をまたぐ」操作が理論名の由来:
- Inter-universal = 宇宙際 = 「宇宙と宇宙の間」
