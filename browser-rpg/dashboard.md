# Loveless Chronicle

転職・スキル・自動戦闘が楽しめるRPGブラウザゲーム。

## 概要

- 種別: game
- 状態: wip
- ファイル: `index.html` (ビルド生成物)
- ビルド: `bash build.sh`

## ソース構成

```
src/
├── css/style.css          # スタイル
├── data/                  # ゲームデータ
│   ├── constants.js       # 定数
│   ├── jobs.js            # 職業
│   ├── skills.js          # スキル
│   ├── equips.js          # 装備
│   ├── items.js           # アイテム
│   ├── monsters.js        # モンスター
│   ├── areas.js           # エリア・マップ
│   ├── shops.js           # ショップ
│   ├── recipes.js         # 合成・鍛冶レシピ
│   └── arena.js           # 闘技場
├── engine/                # ゲームロジック
│   ├── state.js           # 状態管理
│   ├── stats.js           # ステータス計算・成長
│   ├── combat.js          # 戦闘システム
│   ├── economy.js         # 経済(ショップ・銀行・倉庫)
│   ├── crafting.js        # クラフト(鍛冶・合成・強化)
│   ├── navigation.js      # ナビゲーション
│   └── save.js            # セーブ・ロード
├── ui/                    # UI
│   ├── components.js      # 共通コンポーネント
│   ├── screens.js         # 画面レンダリング
│   ├── events.js          # イベントハンドリング
│   └── render.js          # レンダラー
├── plugins/social.js      # ソーシャルプラグイン(スタブ)
└── main.js                # エントリポイント
```

## 機能

- [x] タイトル画面・名前入力・職業選択
- [x] 9種の職業 (下級4 + 上級5)
- [x] 転職システム (ステータス引き継ぎ, 上級職→上級職は不可)
- [x] 隠し成長確率システム
- [x] 装備の成長率ボーナス
- [x] 自動戦闘 (4つの作戦)
- [x] スキルシステム (職業+装備)
- [x] 9エリア・6町のワールドマップ (フィールド移動時は必ず戦闘)
- [x] 5体のボス (ドラゴン, 堕ちたエルフ王, 古城の主, 伝説のプレイヤーたち, 魔神)
- [x] 武器耐久値・修理
- [x] 強化システム (+1~+10, 失敗で破壊)
- [x] 合成システム (レシピ制)
- [x] 鍛冶システム (鉱石→装備)
- [x] 闘技場 (NPC戦)
- [x] 銀行・倉庫
- [x] 宿屋
- [x] セーブ/ロード (localStorage)
- [x] プラグインAPI (ソーシャル拡張用)
- [ ] 酒場チャット (プラグイン)
- [ ] 対人闘技場 (プラグイン)
