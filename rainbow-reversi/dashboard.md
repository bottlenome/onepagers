# 虹色リバーシ

虹色グラデーションの駒で最大4人対戦できるリバーシ。

## 概要

- 種別: game
- 状態: done
- ファイル: `index.html`

## ルール

- 10x10 ボード
- 2〜4人対戦
- 全員挟み（自分の駒で挟んだ間の駒を色問わず自分の色に変える）
- 置けないプレイヤーはパス（全員置けなくなったら終了）
- 初期配置: 中央に各プレイヤー2個ずつ、円形に交互配置（ランダム回転）

## 機能

- [x] 10x10 ボード描画
- [x] 多人数リバーシロジック（全員挟み判定・反転）
- [x] 虹色グラデーション駒（赤・橙・黄・緑）
- [x] 公平な初期配置（中央集中・ランダム回転）
- [x] パス制ターン管理・終局判定
- [x] プレイヤーパネル・スコア表示
- [x] 人数選択画面
