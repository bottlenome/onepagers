# Loveless Chronicle 仕様書

## 概要
転職・スキル・自動戦闘が楽しめる1ページRPGブラウザゲーム。

### 転職ルール
- **下級職転職**: 始まりの町。Lv.1に戻る。ステータス**50%**引き継ぎ（初期値を除いた成長分のみ）
- **上級職転職**: 港町。Lv.15以上必要。**レベルはそのまま**、転職時に**上級職の基本成長値×3のステータスボーナス**が付与される。ただし**上級職から上級職への転職は不可**（一度下級職に戻す必要あり）

### フィールド移動
- フィールド間の移動時、移動先で**必ずモンスターと遭遇**して戦闘になる
- **町の出口はモンスター出現階層に直結**: レベルアップの快適さのため、町を出ると必ずモンスターが出現する階層に出る（例: エルフの里→迷いの森 階層25、最後の街→荒野の道 階層25）

## 成長システム (隠しパラメータ)

レベルアップ時、各ステータスは以下の2段階で上昇する:
1. **確定成長**: 職業の基本成長値(上表)分が必ず上昇
2. **確率成長**: 職業ごとの成長確率(隠し)に基づき追加で+1

## 戦闘システム

### 自動戦闘
- パーティは1人
- SPDが高い方が先制
- 作戦に基づきAIが行動選択

### 作戦
| 作戦 | 説明 |
|------|------|
| がんがんいこうぜ | 最強スキルで攻撃優先 |
| おまかせ | バランスよく戦う |
| いのちだいじに | HP50%以下で回復優先 |
| じゅもんつかうな | 通常攻撃とアイテムのみ |

### ダメージ計算
- 物理: `max(1, ATK × 倍率 - DEF × 0.5) ± 2`
- 魔法: `max(1, MATK × 倍率 - MDEF × 0.5) ± 2`
- 会心: 基本5%で2倍ダメージ

### 経験値
- **プレイヤーLv ≤ モンスターLv** (高階層挑戦): `15 + モンスターLv + 差分 × 2` (ボーナス)
- **プレイヤーLv > モンスターLv** (低階層): `15 + モンスターLv - 差分 × 4` (差5超で0)
- 次レベルまで: `現在Lv × 10`

### モンスターステータス (レベルから自動計算)
- HP: `Lv × 10 + 10`
- ATK: `Lv × 3 + 2`
- DEF/MATK/MDEF: `Lv × 2`
- SPD: `Lv × 2 + 2`

## 武器耐久値

- 武器のみに耐久値あり
- 戦闘1回ごとに耐久値-1
- 耐久値0で武器消滅(ロスト)
- **修理**: 鍛冶屋で修理可能。ただし修理のたびに最大耐久値が5%低下

## オープニングデモバトル

「はじめから」選択時、名前入力の前にLv30騎士vsドラゴンの自動戦闘デモが再生される。

### デモバトルの流れ
1. タイトル画面「はじめから」を押す
2. デモ用Lv30騎士（伝説の勇者）が自動生成される
3. ドラゴン（鉱山 層30ボス）との自動戦闘が開始（最大15ターン）
4. 勝敗問わず「…それは、まだ遠い未来の話。」→ 名前入力画面へ遷移

### 初回格上モンスター遭遇
- ゲーム開始後の最初の探索で、Lv8の「凶暴な○○」と強制エンカウント
- 通常の死亡ペナルティが適用される（デモとは異なり本番）

---

## 死亡ペナルティ

- 戦闘敗北時、所持金の**15%**を喪失する
- 銀行に預けたゴールドは安全（喪失しない）
- 死亡時に銀行預金がある場合「銀行の預金○○Gは無事だ」と表示
- デモバトル中はペナルティなし

## 転移の巻物

消耗品アイテム。探索済みの最深階層へ瞬間移動する。

- 価格: 50G
- 販売場所: 港町アイテム屋
- 使用条件: フィールドにいる時のみ
- 効果: 同エリア内の最深到達階層にワープし、即モンスターと遭遇
- 最深階層が現在階層以下の場合は使用不可

## 装備比較UI

ショップの購入タブで装備品を表示する際、現在装備中のアイテムとの差分を表示する。

- 上昇ステータス: 緑色で `+N` 表示
- 低下ステータス: 赤色で `-N` 表示
- 未装備の場合: 比較なし（通常のステータス表示のみ）

## ソーシャル機能 (プラグイン拡張)

以下の機能はプラグインAPIで拡張可能な設計:
- **酒場** (城): チャットシステム
- **対人闘技場** (城): ユーザーが育てたキャラを登録し、他のユーザーと戦う

プラグインAPI: `PluginAPI.register({ name, facilities, screens })`
