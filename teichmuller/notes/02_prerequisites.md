# 前提知識マップ

IUT理論の理解に関連する数学的概念を、深さ別に整理する。
全てを厳密に理解する必要はない。**レベル1だけでも全体の「筋書き」は追える。**

---

## レベル1: これだけ知っていれば概要が追える

### 素数と素因数分解

- すべての正の整数は素数の積として一意に表せる（算術の基本定理）
- 例: 360 = 2³ × 3² × 5
- ABC予想は「この素因数分解の構造」と「足し算の結果」の関係を述べる

### 楕円曲線（超入門）

- y² = x³ + ax + b の形をした曲線
- 単なる図形ではなく、**曲線上の点同士に「足し算」が定義できる**（群構造）
- 暗号理論（楕円曲線暗号）にも使われている身近な対象
- IUT理論では「数体上の楕円曲線」が出発点

### 数体

- 有理数 Q に代数的な数を付け加えた体（四則演算ができる数の世界）
- 例: Q(√2) = { a + b√2 | a, b ∈ Q }
- 数体は「足し算」と「掛け算」の両方の構造を持つ（環・体の構造）

### 足し算 vs 掛け算の分離

- 普通の数の世界では足し算と掛け算は不可分に絡み合っている
- IUT理論の核心は「これを一度バラして別々に扱う」こと
- 日常の比喩: 色と形が一体になった積み木を、色だけ・形だけに分解するイメージ

## レベル2: 各論文の内容を追うために

### ガロア群

- 方程式の解（根）の間の「対称性」を記述する群
- 例: x² - 2 = 0 の根は ±√2。√2 → -√2 の入れ替えがガロア群の元
- **遠アーベル幾何学**: 幾何学的対象がそのガロア群（基本群）から復元できるという理論
- IUT理論では、環構造を壊した後に遠アーベル幾何学で情報を復元する

### テータ関数

- θ(q) = Σ q^(n²) のような形の関数（qの冪級数）
- 楕円曲線の理論で中心的な役割を果たす
- IUT理論では「**エタールテータ関数**」の特殊値がシータリンクを構成する
- ガウス積分 ∫e^(-x²)dx = √π との深い類似がある

### p進数

- 素数 p に基づく「距離」で数を測る別の世界
- 通常の距離: |1/1000| は小さい
- p進距離: p=2 のとき |1024|₂ は小さい（1024 = 2¹⁰ なので）
- IUT理論では各素数ごとの「局所的な情報」を集めて大域的な結論を出す

### 被覆と基本群

- ある空間を「覆う」ような構造（被覆空間）
- 穴あきドーナツの上を歩くと、被覆空間ではより広い空間を歩くことになる
- 基本群 = ループの同値類がなす群
- エタール基本群 = その代数的・数論的バージョン

## レベル3: 論文の技術的詳細に踏み込むために

### フロベニオイド (Frobenioid)

- 望月が導入した圏論的構造
- 数体の「掛け算的」な部分を抽象化したもの
- 名前の由来: フロベニウス写像 + モノイド

### アーベルの半グラフ (Semi-graphs of Anabelioids)

- 曲線の退化（特殊ファイバー）を組み合わせ論的に記述するための枠組み
- グラフの頂点と辺に「遠アーベル的」な対象を配置したもの

### ホッジ・アラケロフ理論

- 望月のIUT以前の理論（2000年頃）
- 楕円曲線のl等分点上でのテータ関数の値を「ホッジ理論的に」理解する
- この理論の「ガウス極」という技術的障害を乗り越えるためにIUTが生まれた

### 対数殻 (Log-shell)

- p進対数写像の像を少し調整した集合
- 足し算的構造を「掛け算的構造の入れ物」として利用する
- 対数テータ格子の縦方向の矢印（対数リンク）を構成する

### マルチラジアルアルゴリズム

- 「異なる算術的正則構造」の観点からも意味をなすアルゴリズム
- 通常の環論的な議論は一つの環構造に閉じている（ユニラジアル）
- マルチラジアルは複数の環構造をまたぐことができる
- IUT理論の中核的な技術
